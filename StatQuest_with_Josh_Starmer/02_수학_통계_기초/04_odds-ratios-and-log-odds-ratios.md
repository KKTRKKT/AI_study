## 1. 오즈비(Odds Ratio)란?

### 🤔 오즈와 오즈비의 차이

이전 문서에서 오즈를 배웠어요. 오즈는 **한 그룹 안에서** 어떤 사건이 일어날 가능성과 일어나지 않을 가능성의 비율이었어요.

오즈비는 **두 그룹 사이**에서 오즈를 비교하는 거예요.

### 한 줄 정의

> **오즈비(Odds Ratio)란:** 두 그룹의 오즈를 나눈 비율이에요. 한 그룹이 다른 그룹보다 특정 사건이 일어날 가능성이 얼마나 더/덜 높은지를 나타내요.

```
오즈비 = 그룹 1의 오즈 / 그룹 2의 오즈
```

### 일상 비유

> 💡 **비유:** "커피를 마시는 사람"과 "커피를 마시지 않는 사람" 두 그룹에서 각각 불면증 오즈를 구했어요. 이 두 오즈의 비율이 오즈비예요. "커피를 마시면 불면증 오즈가 X배 더 높다"처럼 두 그룹을 비교할 때 쓰는 도구예요.

---

## 2. 실제 예시 — 변이 유전자와 암의 관계

### 데이터 설명

356명의 데이터가 있어요.
- 암 있음: 29명 / 암 없음: 327명
- 변이 유전자 있음: 140명 / 변이 유전자 없음: 216명

이를 2×2 표로 나타내면:

|  | 암 있음 | 암 없음 |
|---|---|---|
| **변이 유전자 있음** | 23 | 117 |
| **변이 유전자 없음** | 6 | 210 |

**질문:** 변이 유전자가 있으면 암에 걸릴 오즈가 더 높을까요?

### 오즈비 계산

**그룹 1 — 변이 유전자 있는 사람:**
```
오즈(변이 유전자 있음) = 23 / 117 ≈ 0.197
```
변이 유전자가 있는 140명 중 암이 있는 23명 vs 암이 없는 117명이에요.

**그룹 2 — 변이 유전자 없는 사람:**
```
오즈(변이 유전자 없음) = 6 / 210 ≈ 0.029
```
변이 유전자가 없는 216명 중 암이 있는 6명 vs 암이 없는 210명이에요.

**오즈비 계산:**
```
오즈비 = (23/117) / (6/210)
       = 0.197 / 0.029
       ≈ 6.88
```

### 해석

오즈비 **6.88**이 의미하는 것: 변이 유전자가 있는 사람은 없는 사람에 비해 암에 걸릴 오즈가 약 **6.88배** 더 높아요.

> 💡 **보충 설명:** 오즈비가 1이면 두 그룹 사이에 차이가 없어요(관계가 없어요). 1보다 크면 그룹 1에서 그 사건이 더 자주 일어나요. 1보다 작으면 그룹 1에서 더 드물게 일어나요.

## 3. 로그 오즈비 — 대칭성 확보

오즈비도 오즈와 같은 비대칭 문제가 있어요.

- 오즈비가 작을 때(관계 없음 쪽): 0에서 1 사이에 몰려 있어요
- 오즈비가 클 때(관계 있음 쪽): 1에서 무한대로 퍼져 있어요

로그를 취하면 대칭이 돼요.

```
로그 오즈비 = log(오즈비) = log(6.88) ≈ 1.93
```

**대칭성 예시:**
- 오즈비 = 2/4 ÷ 3/1 = 0.17 → log(0.17) ≈ **-1.79**
- 오즈비 = 3/1 ÷ 2/4 = 6.0 → log(6.0)  ≈ **+1.79**

정반대 상황의 로그 오즈비가 부호만 다르고 절댓값은 같아요.

## 4. 오즈비는 통계적으로 유의미한가요?

### 🤔 오즈비만으로는 부족해요

오즈비 6.88이 나왔어요. 이 값이 크게 느껴지지만, 혹시 우연히 이런 결과가 나온 건 아닐까요? 통계적으로 의미 있는지를 검정해야 해요.

세 가지 방법을 소개할게요.

---

### ① 피셔 정확 검정 (Fisher's Exact Test)

데이터를 M&M으로 비유해봐요.

- 암 있는 사람(29명) → 빨간 M&M 29개
- 암 없는 사람(327명) → 파란 M&M 327개
- 봉지에서 무작위로 140개를 꺼낼 때 빨간 게 23개 나올 확률은?

이 확률이 아주 작으면 "우연이 아니다 → 통계적으로 유의미하다"는 결론이 나와요.

컴퓨터로 계산하면: **p값 = 0.00001**

이 p값은 "변이 유전자와 암이 관계 없다는 가정이 맞다면 이런 결과가 나올 확률이 0.001%"라는 뜻이에요. 엄청나게 작으니까 관계가 있다고 볼 수 있어요.

---

### ② 카이제곱 검정 (Chi-Square Test)

**아이디어:** "만약 유전자와 암이 관계없다면" 기대되는 값을 계산하고, 실제 관측값과 비교해요.

**기대값 계산 방법:**

전체에서 암 발생 확률 = 29 / 356 ≈ 0.08

"변이 유전자가 암과 관계없다면" 변이 유전자 그룹(140명)에서 기대되는 암 환자 수:
```
기대값 = 0.08 × 140 = 11.2명
```

실제 관측값은 23명이에요. 기대값(11.2)보다 훨씬 많아요.

카이제곱 검정으로 관측값과 기대값의 차이를 검정하면:
- **p값 = 0.00001** (연속성 보정 포함)

피셔 정확 검정과 비슷한 결과가 나왔어요.

---

### ③ 왈드 검정 (Wald Test)

**아이디어:** 로그 오즈비가 0에서 얼마나 멀리 떨어져 있는지를 표준편차 단위로 측정해요.

로그 오즈비가 0이면 → 오즈비가 1이면 → 두 그룹 사이에 차이가 없어요. 로그 오즈비가 0에서 멀수록 더 강한 관계를 나타내요.

**표준편차 추정:**

관측값에서 표준편차를 추정할 수 있어요.
```
표준편차 ≈ √(1/23 + 1/117 + 1/6 + 1/210) ≈ 0.47
```

**이 수식이 말하는 것:**
각 칸의 관측값이 클수록 추정이 안정적이고 표준편차가 작아요. 반대로 칸에 데이터가 적으면 표준편차가 커져요.

**표준화:**
```
Z = 로그 오즈비 / 표준편차
  = 1.93 / 0.47
  ≈ 4.11
```

4.11 표준편차 떨어져 있어요. 일반적으로 2 표준편차 이상이면 p값이 0.05보다 작아요. 4.11이면 매우 강한 증거예요.

계산된 **p값 ≈ 0.00005**

---

### 세 방법 비교

| 검정 방법 | p값 | 특징 |
|---|---|---|
| 피셔 정확 검정 | 0.00001 | 소표본에서 정확해요. 계산이 복잡해요 |
| 카이제곱 (연속성 보정) | 0.00001 | 연속성 보정이 보수적이에요 |
| 카이제곱 (보정 없음) | 0.000004 | 조금 더 관대해요 |
| 왈드 검정 | 0.00005 | 로지스틱 회귀에서 자주 쓰여요. 신뢰 구간도 제공해요 |

세 방법 모두 p값이 0.05보다 훨씬 작아요. 변이 유전자와 암 사이에 통계적으로 유의미한 관계가 있어요.

> 💡 **어떤 방법을 써야 하나요?** 정해진 정답은 없어요. 분야마다 관행이 달라요. 자신의 분야에서 가장 많이 쓰이는 방법을 따르거나, 여러 방법으로 모두 검정해서 일관된 결과가 나오는지 확인하는 것도 좋아요. 경계선 상의 p값이라면 여러 방법을 모두 통과하는지 확인하면 더 신뢰할 수 있어요.

## 5. 오즈비와 R²의 공통점

오즈비(와 로그 오즈비)는 R²(결정계수)와 비슷한 역할을 해요.

- R²: 두 연속형 변수 사이의 관계 강도를 나타내요
- 오즈비: 두 이진 변수(또는 범주형 변수) 사이의 관계 강도를 나타내요

| 값 | 오즈비 해석 |
|---|---|
| = 1 | 관계 없음 (두 그룹 차이 없어요) |
| > 1 | 그룹 1에서 사건이 더 자주 일어나요 |
| < 1 | 그룹 1에서 사건이 더 드물게 일어나요 |
| 6.88 | 그룹 1에서 그 사건이 6.88배 더 자주 일어나요 |

값이 클수록(또는 1과 멀수록) 두 변수 사이의 관계가 강해요.

---

## ⚠️ 주의할 점 / 자주 하는 실수

- **오즈비를 상대 위험도(Relative Risk)와 혼동하지 마세요.** 오즈비와 상대 위험도는 비슷해 보이지만 다른 개념이에요. 드문 사건의 경우엔 비슷한 값이 나오지만, 자주 일어나는 사건에서는 크게 달라져요.
- **오즈비는 인과관계를 증명하지 않아요.** 변이 유전자와 암의 오즈비가 크다고 해서 "변이 유전자가 암을 일으킨다"고 단정할 수 없어요. 상관관계일 뿐이에요.
- **카이제곱 검정은 기대값이 너무 작으면 정확하지 않아요.** 각 칸의 기대값이 5 이상이어야 안정적인 결과가 나와요. 그보다 작으면 피셔 정확 검정을 쓰는 게 좋아요.
- **로그 오즈비 0은 "관계 없음"을 의미해요.** 왈드 검정에서 로그 오즈비가 0에서 멀수록 관계가 강한 거예요.

**Q1.** 오즈(Odds)와 오즈비(Odds Ratio)의 차이를 한 문장으로 설명해보세요.

<details>
<summary>정답 보기</summary>

**정답:** 오즈는 한 그룹 안에서 어떤 사건이 일어날 가능성과 일어나지 않을 가능성의 비율이고, 오즈비는 두 그룹의 오즈를 나눈 비율이에요.

예를 들어 "흡연자의 폐암 오즈"는 오즈이고, "흡연자의 폐암 오즈 / 비흡연자의 폐암 오즈"는 오즈비예요.

</details>

**Q2.** 어떤 연구에서 오즈비가 정확히 1.0이 나왔어요. 어떤 의미인가요?

<details>
<summary>정답 보기</summary>

**정답:** 두 그룹 사이에 관계가 없다는 의미예요.

**설명:** 오즈비 = 그룹1 오즈 / 그룹2 오즈 = 1이면 두 그룹의 오즈가 똑같다는 뜻이에요. 즉, 어떤 그룹에 속하든 그 사건이 일어날 가능성이 동일해요. 두 변수 사이에 아무런 관계가 없는 거예요. 로그 오즈비로 변환하면 log(1) = 0이에요.

</details>

**Q3.** 왈드 검정이 로그 오즈비를 사용하는 이유가 무엇인가요?

<details>
<summary>정답 보기</summary>

**정답:** 로그 오즈비가 정규 분포를 따르기 때문이에요. 정규 분포를 따르면 표준편차 단위로 거리를 계산하고 p값을 구하는 게 쉬워져요.

**설명:** 왈드 검정은 "귀무가설(관계 없음)이 맞다면 로그 오즈비는 0 근처에서 정규 분포를 따른다"는 가정을 이용해요. 실제 계산된 로그 오즈비가 0에서 몇 표준편차 떨어져 있는지 보고 p값을 구해요. 2 표준편차 이상이면 p값 < 0.05로 통계적으로 유의미해요.

</details>

**Q4.** 피셔 정확 검정과 카이제곱 검정 중 어느 상황에서 어느 것을 더 선호하나요?

<details>
<summary>정답 보기</summary>

**정답:** 데이터가 적을 때(각 칸의 기대값이 5 미만)는 피셔 정확 검정을, 데이터가 충분할 때는 카이제곱 검정을 선호해요.

**설명:** 카이제곱 검정은 각 칸에 충분한 데이터가 있어야 정확해요. 기대값이 5 미만인 칸이 있으면 결과가 불안정해져요. 그런 경우 피셔 정확 검정이 더 정확해요. 데이터가 충분하면 두 방법 모두 비슷한 결과를 주는데, 카이제곱 검정이 계산이 더 간단하고 빨라요.

</details>

**Q5.** 로그 오즈비가 -1.5라면 어떤 의미인가요? 그리고 +1.5일 때와 어떤 관계인가요?

<details>
<summary>정답 보기</summary>

**정답:** 로그 오즈비 -1.5는 그룹 1에서 그 사건이 그룹 2보다 더 드물게 일어난다는 의미예요. +1.5와는 크기는 같고 방향이 반대예요.

**설명:** 로그 오즈비 -1.5를 오즈비로 변환하면 e^(-1.5) ≈ 0.22예요. 그룹 1의 오즈가 그룹 2의 22% 수준이라는 뜻이에요. +1.5의 오즈비는 e^(+1.5) ≈ 4.48이에요. 두 그룹을 바꾸면 같은 데이터에서 로그 오즈비 부호만 반대로 바뀌어요. 로그를 쓰는 덕분에 이런 대칭성이 생겨요.

</details>

### 실전 적용 문제

**문제.** 특정 약을 먹은 그룹(200명)과 플라시보를 먹은 그룹(200명)의 회복률 데이터가 있어요.

|  | 회복 | 미회복 |
|---|---|---|
| **약 그룹** | 140 | 60 |
| **플라시보 그룹** | 100 | 100 |

1. 약 그룹의 오즈와 플라시보 그룹의 오즈를 각각 계산해보세요
2. 오즈비를 계산하고 해석해보세요
3. 오즈비가 통계적으로 유의미한지 왈드 검정의 아이디어를 이용해서 판단해보세요 (계산 없이, 방향성만 설명해도 돼요)

**힌트:**
- 오즈 = 회복 수 / 미회복 수
- 오즈비 = 약 그룹 오즈 / 플라시보 그룹 오즈
- 로그 오즈비가 0에서 얼마나 멀리 떨어져 있는지 생각해봐요

<details>
<summary>예시 답안 보기</summary>

**1. 오즈 계산:**
```
약 그룹 오즈    = 140 / 60 ≈ 2.33
플라시보 그룹 오즈 = 100 / 100 = 1.00
```

**2. 오즈비 계산과 해석:**
```
오즈비 = 2.33 / 1.00 = 2.33
```
약을 먹은 그룹이 플라시보를 먹은 그룹보다 회복 오즈가 **2.33배** 높아요. 약이 어느 정도 효과가 있는 것으로 보여요.

로그 오즈비 = log(2.33) ≈ 0.85

**3. 통계적 유의성 판단 (방향성):**
로그 오즈비 0.85는 0에서 어느 정도 떨어져 있어요. 왈드 검정은 이 값이 표준편차 단위로 0에서 얼마나 떨어졌는지 봐요. 각 칸의 수가 충분히 크기 때문에 표준편차 추정이 안정적이고, 0.85가 2 표준편차 이상 떨어져 있는지를 계산하면 통계적 유의성을 판단할 수 있어요. 200명이라는 큰 표본 크기를 고려하면 유의미할 가능성이 높지만, 정확한 계산이 필요해요.

> 정답이 하나가 아닐 수 있어요. 방향이 맞으면 돼요.

</details>

## 🎯 핵심 요약

| 개념 | 한 줄 설명 |
|---|---|
| ⚖️ 오즈비(Odds Ratio) | 두 그룹의 오즈를 나눈 비율이에요. 1이면 관계 없음, 클수록 관계가 강해요 |
| 📐 로그 오즈비 | 오즈비에 로그를 취해서 대칭을 만든 거예요. 0이 중심(관계 없음)이에요 |
| 🧮 피셔 정확 검정 | 소표본에서 정확해요. 계산이 복잡해서 보통 컴퓨터를 써요 |
| 📊 카이제곱 검정 | 관측값 vs 기대값을 비교해요. 각 칸에 충분한 데이터가 있어야 해요 |
| 🔢 왈드 검정 | 로그 오즈비가 0에서 몇 표준편차 떨어져 있는지 계산해요. 로지스틱 회귀에서 자주 써요 |
| 📏 오즈비 해석 | 1 = 관계 없음, >1 = 그룹1이 더 자주 발생, <1 = 그룹1이 더 드물게 발생 |
| 📉 R²와의 공통점 | 둘 다 두 변수 사이 관계의 강도를 나타내는 효과 크기 지표예요 |
