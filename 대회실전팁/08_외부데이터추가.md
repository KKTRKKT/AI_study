## 개요

복잡한 기법 없이 **기본 전략**으로 높은 순위 달성:

1. 외부 데이터 33k 추가
2. LoRA rank를 64로 높임
3. TTA 적용

---

## 학습 설정
```python
@dataclass
class Config:
    checkpoint: str = "unsloth/gemma-2-9b-it-bnb-4bit"
    max_length: int = 1024
    n_splits: int = 800  # 거의 전체 데이터 사용
    fold_idx: int = 0
    lr: float = 1e-5
    warmup_steps: int = 20
    lora_r: int = 64
    lora_alpha: float = 32
    lora_dropout: float = 0.05
```

---

## LoRA 설정

- 대상 모듈: q_proj, k_proj, v_proj, o_proj, gate_proj
- task_type: SEQ_CLS

---

## 추론 핵심 포인트

- **TTA**: 응답 A/B를 바꿔서 예측 후 평균
- max_length를 늘리면 LB 점수가 높아지지만 타임아웃 위험 존재
- 동일한 설정으로도 타임아웃이 발생하거나 안 하거나 불안정
