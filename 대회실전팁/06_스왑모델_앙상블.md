## 솔루션 요약

---

## 특이한 접근법: 스왑 버전 모델

- **TTA 대신**: 응답 A/B를 스왑한 데이터로 별도 모델 훈련
- 원본 모델 + 스왑 모델을 앙상블 (각 모델은 TTA 없이 추론)
- 메모리 효율적이고 성능도 TTA와 유사

---

## 데이터

- Kaggle 공식 학습 데이터
- 33k 추가 데이터
- lmsys-chat-1M에서 Pseudo Label 데이터 생성

### Pseudo Label 데이터 생성
```python
# lmsys-chat-1m에서 영어, 비편집 데이터만 필터
df = df[(df.language == "English") & (~df["redacted"])]
# 각 프롬프트에서 다른 모델 응답 2개씩 샘플링
# → 약 45k 추가 학습 데이터 생성
```
