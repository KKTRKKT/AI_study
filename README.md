## 🗺️ 전체 개념 지도

```
머신러닝 (ML)
│
├── 지도 학습 (Supervised Learning) → 정답(Label)이 있음
│   ├── 회귀 (Regression) → 숫자를 예측
│   │   └── 평가: MSE, RMSE, R²
│   └── 분류 (Classification) → 범주를 예측
│       └── 평가: Accuracy, Precision, Recall, F1-Score
│
├── 비지도 학습 (Unsupervised Learning) → 정답(Label)이 없음
│   ├── 군집화 (Clustering) → 비슷한 것끼리 묶기
│   │   ├── K-means Clustering
│   │   └── 계층적 군집화 (Hierarchical Clustering)
│   └── 차원 축소, 이상치 탐지 등
│
└── 모델 평가 & 검증
    ├── 과적합 (Overfitting) vs 과소적합 (Underfitting)
    ├── 훈련 오류 vs 테스트 오류
    ├── 검증셋 방법 (Hold-out)
    ├── K-fold 교차 검증
    └── LOOCV
```

## 💡 핵심 개념 한줄 요약

| 개념 | 한줄 요약 |
|------|---------|
| **Feature** | 모델에 넣는 입력값 (예측의 근거) |
| **Label** | 모델이 맞춰야 할 정답값 |
| **지도 학습** | Feature + Label로 예측 규칙을 배우는 방법 |
| **비지도 학습** | Label 없이 데이터 구조를 찾는 방법 |
| **회귀** | 숫자를 예측 (집값, 온도, 점수) |
| **분류** | 범주를 예측 (스팸/정상, 질병 유무) |
| **MSE** | 예측 오차의 제곱 평균 (작을수록 좋음) |
| **R²** | 모델이 데이터 변동을 설명하는 비율 (1에 가까울수록 좋음) |
| **정밀도** | 양성 예측 중 실제 양성의 비율 |
| **재현율** | 실제 양성 중 잡아낸 비율 |
| **F1-Score** | 정밀도와 재현율의 균형 지표 |
| **과적합** | 훈련 데이터만 잘 맞추고 새 데이터는 못 맞추는 현상 |
| **K-fold CV** | 데이터를 K개로 나눠 번갈아 검증하는 방법 |
| **K-means** | K개 군집으로 나누는 클러스터링 알고리즘 |
| **계층적 군집** | 트리 구조로 군집을 합치는 알고리즘 |
