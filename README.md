## 🗺️ 전체 개념 지도

```
머신러닝 (ML)
│
├── 지도 학습 (Supervised Learning) → 정답(Label)이 있음
│   ├── 회귀 (Regression) → 숫자를 예측
│   │   ├── 단순선형회귀: y = β₀ + β₁x
│   │   ├── 다중선형회귀: y = β₀ + β₁x₁ + β₂x₂ + ...
│   │   ├── 추정: 최소제곱법 (RSS 최소화)
│   │   └── 평가: MSE, RMSE, R²
│   │
│   └── 분류 (Classification) → 범주를 예측
│       ├── 이진 분류 (Yes/No, 스팸/정상)
│       ├── 로지스틱 회귀 (시그모이드 함수)
│       ├── 추정: MLE (최대우도추정)
│       └── 평가: Accuracy, Precision, Recall, F1-Score
│
├── 비지도 학습 (Unsupervised Learning) → 정답(Label)이 없음
│   ├── 군집화 (Clustering)
│   │   ├── K-means Clustering
│   │   └── 계층적 군집화 (Hierarchical Clustering)
│   └── 차원 축소, 이상치 탐지 등
│
├── 신경망 (Neural Network)
│   ├── Shallow Network (1개 은닉층)
│   │   ├── 활성화 함수 (ReLU)
│   │   └── Hidden Units
│   └── Deep Network (여러 은닉층)
│       ├── 층을 쌓으면 표현력 기하급수적 증가
│       └── 행렬 표기: hₖ = a[βₖ₋₁ + Ωₖ₋₁hₖ₋₁]
│
└── 모델 학습 알고리즘
    ├── 손실함수 (Loss Function) → 오차 측정
    ├── 경사하강법 (Gradient Descent) → 파라미터 업데이트
    ├── SGD (확률적 경사하강법) → 미니배치로 효율화
    ├── 역전파 (Backpropagation) → 연쇄법칙으로 기울기 계산
    │
    └── 모델 평가 & 검증
        ├── 과적합 (Overfitting) vs 과소적합 (Underfitting)
        ├── 훈련 오류 vs 테스트 오류
        ├── 검증셋 방법 (Hold-out)
        ├── K-fold 교차 검증
        └── LOOCV
```

## 💡 핵심 개념 한줄 요약

### 머신러닝 기초

| 개념 | 한줄 요약 |
|------|---------|
| **Feature** | 모델에 넣는 입력값 (예측의 근거) |
| **Label** | 모델이 맞춰야 할 정답값 |
| **지도 학습** | Feature + Label로 예측 규칙을 배우는 방법 |
| **비지도 학습** | Label 없이 데이터 구조를 찾는 방법 |
| **회귀** | 숫자를 예측 (집값, 온도, 점수) |
| **분류** | 범주를 예측 (스팸/정상, 질병 유무) |
| **MSE** | 예측 오차의 제곱 평균 (작을수록 좋음) |
| **R²** | 모델이 데이터 변동을 설명하는 비율 (1에 가까울수록 좋음) |
| **정밀도** | 양성 예측 중 실제 양성의 비율 |
| **재현율** | 실제 양성 중 잡아낸 비율 |
| **F1-Score** | 정밀도와 재현율의 균형 지표 |
| **과적합** | 훈련 데이터만 잘 맞추고 새 데이터는 못 맞추는 현상 |
| **K-fold CV** | 데이터를 K개로 나눠 번갈아 검증하는 방법 |
| **K-means** | K개 군집으로 나누는 클러스터링 알고리즘 |
| **계층적 군집** | 트리 구조로 군집을 합치는 알고리즘 |

### 선형회귀 & 분류

| 개념 | 한줄 요약 |
|------|---------|
| **최소제곱법** | 잔차의 제곱합(RSS)을 최소화해서 최적 직선을 찾는 방법 |
| **RSS** | 잔차(실제-예측)를 제곱해서 합한 값. 작을수록 좋은 모델 |
| **잔차** | 실제값과 예측값의 차이 |
| **다중선형회귀** | 여러 개의 독립변수로 Y를 동시에 예측하는 모델 |
| **p-value** | 이 결과가 우연일 확률. 0.05 미만이면 유의미 |
| **시그모이드** | 어떤 값도 0~1 사이로 눌러주는 S자 함수 |
| **오즈(Odds)** | 성공 확률 / 실패 확률 |
| **로짓** | 오즈에 로그를 취한 값. 범위가 -∞~+∞로 선형회귀처럼 됨 |
| **MLE** | 데이터를 가장 잘 설명하는 파라미터를 찾는 방법 |
| **상관관계≠인과관계** | 두 변수가 함께 변해도 한쪽이 원인이 아닐 수 있음 |

### 신경망 & 딥러닝

| 개념 | 한줄 요약 |
|------|---------|
| **ReLU** | 음수는 0, 양수는 그대로 통과시키는 활성화 함수 |
| **Hidden Unit** | 신경망 내부에서 특징을 추출하는 유닛 |
| **Shallow Network** | 은닉층이 1개인 신경망 |
| **Deep Network** | 은닉층이 여러 개인 신경망 |
| **보편적 근사 정리** | Hidden Unit이 충분하면 어떤 함수든 근사 가능 |
| **손실함수** | 모델이 얼마나 잘못 예측하는지를 숫자로 나타낸 함수 |
| **경사하강법** | 손실이 줄어드는 방향으로 파라미터를 조금씩 이동 |
| **학습률(α)** | 파라미터를 한 번에 얼마나 이동할지 결정하는 값 |
| **SGD** | 전체 데이터 대신 미니배치로 기울기를 근사하는 방법 |
| **역전파** | 연쇄법칙으로 모든 층의 기울기를 한 번에 계산하는 알고리즘 |
| **연쇄법칙** | 합성함수의 미분 = 바깥 함수 미분 × 안쪽 함수 미분 |
| **Convex** | 전역 최솟값이 하나인 함수 (최적화가 쉬움) |
| **Non-convex** | 지역 최솟값이 여러 개인 함수 (딥러닝 손실함수) |
