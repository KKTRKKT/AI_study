## 1. 로지스틱 회귀는 GLM(일반화 선형 모델)이에요

### 🤔 선형 모델과 어떤 관계인가요?

로지스틱 회귀는 **일반화 선형 모델(Generalized Linear Model, GLM)** 의 한 종류예요.

- 선형 회귀 → GLM의 가장 기본 형태
- t-검정, ANOVA → 선형 모델로 표현 가능
- 로지스틱 회귀 → 이 모든 것의 확장 버전

이미 선형 모델을 알고 있다면, 로지스틱 회귀 계수도 비슷한 방식으로 이해할 수 있어요. 핵심 차이는 단 하나: **계수의 단위가 로그 오즈**예요.

---

## 2. y축 변환이 핵심이에요

### 선형 회귀의 y축 vs 로지스틱 회귀의 y축

| | 선형 회귀 | 로지스틱 회귀 |
|---|---|---|
| y축 범위 | -∞ ~ +∞ (어떤 숫자든 가능) | 0 ~ 1 (확률) |
| 문제 | 없음 | 직선을 맞추기 어려움 |
| 해결책 | - | y축을 로그 오즈로 변환 |

### 변환 과정

확률(p)을 로그 오즈로 바꾸는 함수가 **로지트 함수(Logit Function)** 예요.

```
로그 오즈 = logit(p) = log(p / (1-p))
```

**이 수식이 말하는 것:**
- p = 0.5일 때 → 로그 오즈 = log(1) = **0** (중심점)
- p → 1일 때 → 로그 오즈 → **+∞**
- p → 0일 때 → 로그 오즈 → **-∞**

이 변환 후 y축이 -∞~+∞ 범위가 되어서, 선형 회귀처럼 직선을 맞출 수 있어요.

> 💡 **핵심:** 로지스틱 함수의 구불구불한 S자 곡선은 **확률 스케일**에서 보이는 모습이에요. 하지만 내부적으로는 **로그 오즈 스케일**에서 직선을 맞추는 거예요. 계수는 이 로그 오즈 스케일의 값으로 표현돼요.

## 3. 연속형 변수의 계수 — 몸무게로 비만 예측

### 로지스틱 회귀 직선 방정식

로그 오즈 스케일에서 직선:
```
log odds(비만) = 절편 + 기울기 × 몸무게
              = -3.476 + 1.825 × 몸무게
```

이 방정식의 두 계수를 하나씩 해석해볼게요.

### ① 절편 (y축 절편): -3.476

**의미:** 몸무게가 0일 때 비만의 로그 오즈예요.

```
로그 오즈 = -3.476
→ 오즈 = e^(-3.476) ≈ 0.031
→ "몸무게 0인 생쥐는 비만이 아닐 가능성이 압도적으로 높아요"
```

당연한 이야기예요. 몸무게가 없으면 비만일 수가 없죠.

**왈드 검정으로 유의미성 확인:**

```
z-값 = 추정값 / 표준오차 = -3.476 / (표준오차)
```

z-값의 절댓값이 2보다 작으면 통계적으로 유의미하지 않아요.
z-값이 1.47이라서 이 절편은 통계적으로 유의미하지 않아요 (p-값 > 0.05). 표본이 워낙 작아서 예상되는 결과예요.

### ② 기울기(Slope): 1.825

**의미:** 몸무게가 1단위 증가할 때마다 비만의 로그 오즈가 1.825 증가해요.

```
로그 오즈 증가량 = 1.825
→ 오즈 배율 = e^(1.825) ≈ 6.2배
→ "몸무게가 1단위 오를 때마다 비만 오즈가 약 6.2배 늘어나요"
```

**왈드 검정 결과:**
기울기도 작은 표본 때문에 이 예시에서는 통계적으로 유의미하지 않게 나왔어요. 하지만 실제 더 많은 데이터가 있으면 유의미해질 수 있어요.

> 💡 **선형 회귀와 비교:** 선형 회귀의 기울기는 "x가 1 증가할 때 y가 얼마나 증가하는가"예요. 로지스틱 회귀의 기울기는 "x가 1 증가할 때 로그 오즈가 얼마나 증가하는가"예요.

## 4. 이산형 변수의 계수 — 유전자 변이와 비만

### 상황 설정

- 정상 유전자 생쥐(Normal): 일부는 비만, 일부는 정상
- 변이 유전자 생쥐(Mutant): 일부는 비만, 일부는 정상

"변이 유전자가 비만 오즈를 높이는가?"를 로지스틱 회귀로 검정해요.

### t-검정과의 유사성

이 분석은 선형 모델로 t-검정을 하는 것과 구조가 동일해요.

선형 모델에서:
```
y = μ_대조군 + Δ(변이 효과) × (변이군이면 1, 아니면 0)
```

로지스틱 회귀에서:
```
log odds(비만) = log odds_정상군 + log(오즈비) × (변이군이면 1, 아니면 0)
```

### 계수 계산 예시

데이터:
- 정상 유전자 생쥐: 비만 2마리, 정상 9마리
- 변이 유전자 생쥐: 비만 7마리, 정상 3마리

**절편 (정상군의 로그 오즈):**
```
정상군 오즈 = 비만 수 / 정상 수 = 2/9
로그 오즈_정상군 = log(2/9) ≈ -1.504
```

**변이군 계수 (로그 오즈비):**
```
변이군 오즈 = 7/3
오즈비 = (7/3) / (2/9) = (7/3) × (9/2) = 10.5
로그 오즈비 = log(10.5) ≈ 2.351
```

**최종 방정식:**
```
log odds(비만) = -1.504 + 2.351 × (변이군이면 1, 아니면 0)
```

### 계수 해석

- **절편 (-1.504):** 정상 유전자 생쥐의 비만 로그 오즈
- **변이 계수 (2.351):** 로그 오즈비예요. 변이 유전자가 있으면 비만 오즈가 e^2.351 ≈ 10.5배 높아요

**왈드 검정 결과:**
- 절편의 z-값: -1.9 (절댓값 < 2, 유의미하지 않음)
- 변이 계수의 z-값: > 2 (**유의미함!** p < 0.05)

변이 유전자는 비만과 통계적으로 유의미한 관계가 있어요.

## 5. 조합 사용 — 여러 변수를 함께 쓰기

### 다중 로지스틱 회귀

선형 모델에서 회귀, t-검정, ANOVA, 다중 회귀 등 다양한 분석을 할 수 있었던 것처럼, 로지스틱 회귀도 동일하게 확장돼요.

```
log odds(비만) = 절편 + b₁ × 몸무게 + b₂ × (변이군이면 1, 아니면 0) + ...
```

단, 기억해야 할 것은 **모든 계수가 로그 오즈 단위**라는 거예요.

> 🎯 **핵심:** 로지스틱 회귀는 선형 모델과 동일한 구조예요. 다중 회귀, ANOVA 등 모든 것이 가능해요. 단 하나의 차이는 계수가 로그 오즈 스케일로 표현된다는 것이에요.

---

## 6. 왈드 검정 요약

왈드 검정은 각 계수가 통계적으로 유의미한지(0과 다른지) 확인하는 방법이에요.

```
z-값 = 추정 계수 / 표준오차
```

- z-값 절댓값 > 2 → p-값 < 0.05 → **통계적으로 유의미해요**
- z-값 절댓값 < 2 → p-값 > 0.05 → **유의미하지 않아요** (이 변수는 큰 도움이 안 돼요)

> 💡 **보충 설명:** 왈드 검정은 오즈비/로그 오즈비 문서에서 배운 내용과 동일한 원리예요. 로그 오즈비가 정규 분포를 따르기 때문에, 0으로부터의 표준편차 거리로 p-값을 구할 수 있어요.

---

## ⚠️ 주의할 점 / 자주 하는 실수

- **계수 단위를 혼동하지 마세요.** 로지스틱 회귀 계수는 로그 오즈예요. "기울기 = 1.825"는 "몸무게가 1 오를 때 비만 확률이 1.825 오른다"는 게 아니에요. 로그 오즈가 1.825 오른다는 뜻이에요.
- **계수를 확률로 직접 해석하면 안 돼요.** 로그 오즈를 확률로 변환하려면 로지스틱 함수를 거쳐야 해요.
- **표본이 작으면 유의미하지 않은 결과가 나올 수 있어요.** 실제 효과가 있어도 표본이 너무 작으면 통계적으로 유의미하게 나오지 않아요.
- **로그 오즈비는 방향을 포함해요.** 양수 → 그 변수가 있으면 오즈가 증가, 음수 → 그 변수가 있으면 오즈가 감소예요.

**Q1.** 로지스틱 회귀에서 y축을 "확률"에서 "로그 오즈"로 변환하는 이유는 무엇인가요?

<details>
<summary>정답 보기</summary>

**정답:** 확률의 범위는 0~1로 제한되어 있어서 직선을 맞추기 어려워요. 로그 오즈로 변환하면 범위가 -∞~+∞가 되어서 직선을 맞출 수 있어요.

**설명:** 선형 회귀의 가장 큰 장점은 직선을 쉽게 최적화할 수 있다는 거예요. 이 장점을 로지스틱 회귀에서도 활용하려면 y축의 범위를 제한 없이 만들어야 해요. 로지트 함수(logit)로 확률을 로그 오즈로 바꾸면 이게 가능해요.

</details>

**Q2.** 연속형 변수의 기울기 계수가 1.825라면 어떻게 해석하나요?

<details>
<summary>정답 보기</summary>

**정답:** x변수가 1단위 증가할 때마다 결과의 로그 오즈가 1.825 증가해요. 오즈로 변환하면 e^1.825 ≈ 6.2배 증가해요.

**설명:** 계수가 양수면 x가 커질수록 대상 사건의 오즈가 증가해요. 음수면 감소해요. "로그 오즈 1.825 증가"를 확률의 언어로 표현하려면 직접 계산이 필요해요. x의 특정 값에서의 확률은 로지스틱 함수에 넣어서 구해야 해요.

</details>

**Q3.** 이산형 변수에서 로지스틱 회귀의 두 번째 계수(오프셋)가 로그 오즈비인 이유는 무엇인가요?

<details>
<summary>정답 보기</summary>

**정답:** 두 번째 계수는 변이군 로그 오즈에서 대조군 로그 오즈를 뺀 값이고, 이게 수학적으로 두 오즈의 비율(오즈비)의 로그예요.

**설명:** log(A) - log(B) = log(A/B)라는 로그의 성질 때문이에요.
- 절편: log(대조군 오즈) = log(2/9)
- 두 번째 계수: log(변이군 오즈) - log(대조군 오즈) = log[(7/3) / (2/9)] = log(오즈비)

이 계수가 바로 로그 오즈비예요. e를 거듭제곱하면 오즈비가 나와요.

</details>

---

**Q4.** 왈드 검정의 z-값이 2.3이라면 이 계수는 통계적으로 유의미한가요?

<details>
<summary>정답 보기</summary>

**정답:** 네, 유의미해요. z-값 절댓값이 2보다 크면 p-값 < 0.05예요.

**설명:** 왈드 검정은 "이 계수가 0이라면 z-값이 이만큼 클 확률이 얼마나 되나?"를 계산해요. z-값 절댓값이 약 2 이상이면 0에서 2 표준편차 이상 떨어진 것이고, 이는 p-값 < 0.05에 해당해요. 일반적으로 p < 0.05를 통계적 유의미성의 기준으로 써요.

</details>

---

**Q5.** 로지스틱 회귀가 "일반화 선형 모델(GLM)"의 한 종류라는 것이 실용적으로 어떤 의미인가요?

<details>
<summary>정답 보기</summary>

**정답:** 선형 회귀, t-검정, ANOVA에서 배운 모든 개념(다중 회귀, 변수 추가/제거, 모델 비교 등)을 로지스틱 회귀에도 그대로 적용할 수 있어요.

**설명:** 로지스틱 회귀도 디자인 행렬을 쓰고, 여러 변수를 동시에 사용하고, 모델을 비교할 수 있어요. 다중 로지스틱 회귀, 로지스틱 ANCOVA 등이 모두 가능해요. 유일한 차이는 계수가 로그 오즈 단위로 표현된다는 것뿐이에요.

</details>

### 실전 적용 문제

**문제.** 심장 질환 예측 로지스틱 회귀를 돌렸더니 이런 결과가 나왔어요.

```
계수:
절편:       -2.10  (표준오차: 0.50,  z-값: -4.20)
나이:        0.05  (표준오차: 0.02,  z-값:  2.50)
흡연 여부:   1.30  (표준오차: 0.40,  z-값:  3.25)
```

1. 나이 계수(0.05)를 해석해보세요
2. 흡연 여부 계수(1.30)를 해석해보세요 (흡연자 = 1, 비흡연자 = 0으로 코딩했어요)
3. 세 계수 중 통계적으로 유의미한 것은 무엇인가요? z-값을 기준으로 판단해봐요

**힌트:**
- 연속형 변수 계수: "x가 1 증가할 때 로그 오즈가 ___만큼 변해요"
- 이산형 변수 계수: "로그 오즈비예요. e^계수 = 오즈비예요"
- z-값 절댓값 > 2 → 유의미

<details>
<summary>예시 답안 보기</summary>

**1. 나이 계수(0.05) 해석:**
나이가 1살 증가할 때마다 심장 질환의 로그 오즈가 0.05 증가해요. 오즈로 변환하면 e^0.05 ≈ 1.05배 증가해요. 즉, 1살 더 나이를 먹을 때마다 심장 질환 오즈가 약 5% 높아져요.

**2. 흡연 여부 계수(1.30) 해석:**
이건 로그 오즈비예요. 흡연자의 심장 질환 오즈가 비흡연자보다 e^1.30 ≈ 3.67배 높아요. 흡연이 심장 질환 위험을 약 3.7배 높이는 효과가 있어요.

**3. 통계적 유의미성:**
- 절편: z = -4.20, 절댓값 = 4.20 > 2 → **유의미** ✅
- 나이: z = 2.50, 절댓값 = 2.50 > 2 → **유의미** ✅
- 흡연 여부: z = 3.25, 절댓값 = 3.25 > 2 → **유의미** ✅

세 계수 모두 통계적으로 유의미해요.

> 정답이 하나가 아닐 수 있어요. 방향이 맞으면 돼요.

</details>

## 🎯 핵심 요약

| 개념 | 한 줄 설명 |
|---|---|
| 🔄 y축 변환 | 확률(0~1) → 로그 오즈(-∞~+∞)로 변환해서 직선 맞추기가 가능해요 |
| 📐 계수 단위 | 로지스틱 회귀의 모든 계수는 로그 오즈 단위예요 |
| 📈 연속형 계수 | "x가 1 증가할 때 로그 오즈가 얼마나 변하는가"예요 |
| ⚖️ 이산형 계수 | 로그 오즈비예요. e를 거듭제곱하면 오즈비가 나와요 |
| 🔢 왈드 검정 | z = 추정값/표준오차. 절댓값 > 2이면 유의미(p < 0.05)예요 |
| 🔗 GLM | 로지스틱 회귀는 GLM이에요. 다중 회귀, ANOVA 개념이 모두 적용돼요 |
