## 용어 정리

| 용어 | 원문 | 한 줄 설명 |
|---|---|---|
| 포화 모델 | Saturated Model | 데이터 포인트마다 하나의 파라미터를 가진 최대 복잡 모델이에요 |
| 영 모델 | Null Model | 파라미터 하나(전체 평균)만 가진 가장 단순한 모델이에요 |
| 제안 모델 | Proposed Model | 실제로 사용하려는 모델이에요. 영 모델과 포화 모델 사이에 있어요 |
| 잔차 이탈도 | Residual Deviance | 포화 모델과 제안 모델의 로그 우도 차이에서 나온 이탈도예요 |
| 영 이탈도 | Null Deviance | 포화 모델과 영 모델의 로그 우도 차이에서 나온 이탈도예요 |
| 이탈도 | Deviance | -2 × 로그 우도 차이로 계산한 카이제곱 통계량이에요 |

## 1. 세 가지 모델 — 스펙트럼으로 생각해요

### 🤔 왜 포화 모델이 필요한가요?

로지스틱 회귀에서 R²를 계산할 때, 이전 문서에서는 두 모델만 비교했어요.

```
수평선 (나쁜 모델) vs 최적 로지스틱 곡선 (좋은 모델)
```

그런데 다른 GLM(일반화 선형 모델)에서는 "가장 좋은 모델이 무엇인지"에 대한 고정된 기준이 없어요. 로지스틱 회귀에서는 LL적합이 0에 가까울수록 좋다고 했지만, 다른 모델에서는 다를 수 있어요.

이 문제를 해결하기 위해 **포화 모델**을 도입해요. 포화 모델은 "이론적으로 가능한 최고의 모델"이에요.

### 세 가지 모델의 스펙트럼

```
영 모델(최악) ←────────────────────────→ 포화 모델(최고)
     │               제안 모델               │
  파라미터 1개   (파라미터 k개)       파라미터 n개
  (전체 평균)    (우리가 쓸 모델)    (데이터당 1개)
```

- **영 모델(Null Model):** 파라미터 1개. 전체 데이터의 평균만 사용해요.
- **제안 모델(Proposed Model):** 우리가 실제로 사용하려는 모델이에요.
- **포화 모델(Saturated Model):** 데이터 포인트마다 파라미터 1개. 이론적으로 최고의 적합도예요.

> 💡 **일상 비유:** 영 모델은 "반 전체 평균 성적으로 모든 학생의 성적을 예측"하는 것이에요. 포화 모델은 "각 학생의 실제 성적을 그냥 그대로 쓰는" 것이에요. 제안 모델은 "이름, 공부 시간으로 성적을 예측"하는 중간 어딘가예요.

## 2. 예시: 세 모델의 우도 비교

생쥐 6마리 몸무게 데이터로 예시를 들어볼게요.

### 영 모델 (파라미터 1개)

전체 평균 하나의 정규 분포로 데이터를 설명해요.

```
영 모델 우도 = 0.03
영 모델 로그 우도 = log(0.03) = -3.51
```

### 제안 모델 (파라미터 2개)

두 개의 다른 평균으로 데이터를 설명해요 (예: 두 그룹의 평균).

```
제안 모델 우도 = 3.57
제안 모델 로그 우도 = log(3.57) = 1.27
```

### 포화 모델 (파라미터 n개 = 데이터당 1개)

각 데이터 포인트에 완벽히 맞는 모델이에요.

```
포화 모델 우도 = 1291.5
포화 모델 로그 우도 = log(1291.5) = 7.16
```

**우리가 원하는 것:** 제안 모델의 우도가 영 모델보다 훨씬 높고, 포화 모델에 가까울수록 좋아요.

---

## 3. 포화 모델 없이 R²를 계산하면?

이전 문서의 R² 공식을 포화 모델 없이 적용하면:

```
R²(포화 모델 제외) = 1 - (LL제안 / LL영)
                  = 1 - (1.27 / -3.51)
                  = 1 - (-0.36)
                  = 1.36
```

**R² = 1.36?** R²는 0~1 사이여야 하는데 이상해요! 이게 포화 모델이 필요한 이유예요.

---

## 4. 포화 모델이 있는 올바른 R² 공식

```
R² = 1 - (LL제안 - LL포화) / (LL영 - LL포화)
   = 1 - (1.27 - 7.16) / (-3.51 - 7.16)
   = 1 - (-5.89 / -10.67)
   = 1 - 0.552
   = 0.45
```

올바른 R² = 0.45! 0~1 사이에 있어요. ✅

**이 수식이 말하는 것:**
- 분자: 제안 모델이 포화 모델보다 얼마나 나쁜가 (이탈 정도)
- 분모: 영 모델이 포화 모델보다 얼마나 나쁜가 (최대 이탈 정도)
- R²: "우리 모델이 얼마나 이 이탈을 줄였는가"의 비율

---

## 5. 이탈도(Deviance) — 핵심 개념

### 잔차 이탈도 (Residual Deviance)

제안 모델이 포화 모델에 비해 얼마나 "이탈"했는지를 나타내요.

```
잔차 이탈도 = 2 × (LL포화 - LL제안)
```

"2를 곱하는" 이유는 이 차이가 카이제곱 분포를 따르게 만들기 위해서예요.

자유도 = (포화 모델 파라미터 수) - (제안 모델 파라미터 수)

예시:
- 포화 모델: 6개 파라미터
- 제안 모델: 2개 파라미터
- 자유도 = 6 - 2 = **4**

```
잔차 이탈도 = 2 × (7.16 - 1.27) = 2 × 5.89 = 11.78
```

카이제곱 분포(자유도 4)에서 p-값을 구하면: **p = 0.02**

> 💡 **해석:** p = 0.02는 제안 모델과 포화 모델이 유의미하게 다르다는 뜻이에요. 제안 모델이 포화 모델만큼 좋지 않다는 것을 의미해요. (사실 이 통계량 자체의 해석은 논쟁적이에요. 더 중요한 건 영 이탈도와의 차이예요.)

### 영 이탈도 (Null Deviance)

영 모델이 포화 모델에 비해 얼마나 이탈했는지를 나타내요.

```
영 이탈도 = 2 × (LL포화 - LL영)
         = 2 × (7.16 - (-3.51))
         = 2 × 10.67
         = 21.34
```

자유도 = 6 - 1 = **5**

카이제곱 분포(자유도 5)에서 p-값: **p ≈ 0** → 영 모델과 포화 모델은 유의미하게 달라요.

---

## 6. 이탈도 차이로 p-값 구하기

### 핵심 공식

두 이탈도의 차이가 R²에 대한 p-값을 줘요.

```
χ² = 영 이탈도 - 잔차 이탈도
   = 21.34 - 11.78
   = 9.56
```

자유도 = (제안 모델 파라미터) - (영 모델 파라미터) = 2 - 1 = **1**

카이제곱 분포(자유도 1)에서 p-값: **p = 0.002**

이 p-값이 "우리 모델이 영 모델보다 유의미하게 더 좋은가"에 대한 답이에요.

> 💡 **보충 설명:** 이전 R² 문서에서 배운 `χ² = 2×(LL적합 - LL전체확률)` 공식과 같은 결과예요. 두 방법이 동일해요. 이탈도 방법이 더 자주 출력으로 나오기 때문에 알아두면 유용해요.

---

## 7. 로지스틱 회귀에서 포화 모델을 무시할 수 있는 이유

로지스틱 회귀는 특별한 경우예요.

로지스틱 회귀의 포화 모델은 모든 데이터를 완벽히 맞추는 곡선이에요. 이때:
- 비만인 생쥐: 예측 확률 = 1 → 우도 = 1 → log(1) = **0**
- 비만 아닌 생쥐: 예측 확률 = 0 → 우도 = 1 → log(1) = **0**
- **LL포화 = 0 + 0 + ... = 0**

포화 모델 로그 우도가 항상 0이기 때문에, 공식이 단순해져요.

```
R² = 1 - (LL제안 - 0) / (LL영 - 0)
   = 1 - LL제안 / LL영
```

이전 R² 문서의 공식과 동일해요! 로지스틱 회귀에서는 포화 모델이 자동으로 제거돼요.

마찬가지로:
- 잔차 이탈도 = 2 × (0 - LL제안) = -2 × LL제안
- 영 이탈도 = 2 × (0 - LL영) = -2 × LL영

---

## 8. R 출력에서 이탈도 읽기

R에서 로지스틱 회귀를 돌리면 이런 출력이 나와요.

```
Null deviance:     85.92  on 296  degrees of freedom
Residual deviance: 50.34  on 290  degrees of freedom
```

이걸 해석하면:
- **Null deviance = 85.92:** 영 이탈도예요. 영 모델이 포화 모델과 비교한 이탈도예요.
- **Residual deviance = 50.34:** 잔차 이탈도예요. 제안 모델이 포화 모델과 비교한 이탈도예요.
- **Deviance 차이 = 85.92 - 50.34 = 35.58:** 이걸 카이제곱 분포로 검정하면 p-값이 나와요.
- **자유도 차이 = 296 - 290 = 6:** 제안 모델에 추가된 파라미터 수예요.

---

## ⚠️ 주의할 점 / 자주 하는 실수

- **잔차 이탈도(Residual Deviance)가 작을수록 좋은 모델이에요.** 포화 모델과 가까울수록 이탈이 적어요.
- **영 이탈도(Null Deviance)는 고정값이에요.** 어떤 제안 모델을 쓰든 동일해요. 데이터와 영 모델이 정해지면 바뀌지 않아요.
- **이탈도 검정은 모델이 "내포(nested)"되어야 해요.** 제안 모델이 영 모델의 확장 버전이어야 해요.
- **잔차 이탈도만으로 모델 품질을 판단하면 안 돼요.** 영 이탈도와의 차이, p-값을 함께 봐야 해요.

**Q1.** 포화 모델(Saturated Model)이란 무엇이고, 왜 "이론적 최고 모델"인가요?

<details>
<summary>정답 보기</summary>

**정답:** 포화 모델은 데이터 포인트마다 파라미터 하나를 가진 모델이에요. 데이터 수만큼 파라미터가 있어서 모든 데이터를 완벽히 설명할 수 있어요.

**설명:** 데이터가 6개라면 포화 모델은 파라미터 6개를 가져요. 각 파라미터가 각 데이터 포인트에 정확히 맞춰지기 때문에 오차(잔차)가 0이에요. 이게 이론적으로 가능한 최고의 적합도예요. 실제로 이 모델을 쓰면 과적합이 되지만, R² 계산의 "상한선" 기준으로 유용해요.

</details>

**Q2.** 영 모델(Null Model)과 영 이탈도(Null Deviance)의 관계를 설명해보세요.

<details>
<summary>정답 보기</summary>

**정답:** 영 이탈도는 영 모델이 포화 모델과 비교해서 얼마나 "이탈"했는지를 나타내요. 2 × (LL포화 - LL영)으로 계산해요.

**설명:** 영 모델은 파라미터 1개(전체 평균)만 가진 가장 단순한 모델이에요. 포화 모델은 완벽한 적합도예요. 두 모델의 로그 우도 차이에 2를 곱하면 영 이탈도가 나와요. 영 이탈도는 "우리 데이터에서 얼마나 더 개선될 수 있는지"의 기준이 돼요.

</details>

**Q3.** 로지스틱 회귀에서 포화 모델의 로그 우도가 항상 0인 이유는 무엇인가요?

<details>
<summary>정답 보기</summary>

**정답:** 로지스틱 회귀의 포화 모델은 데이터를 완벽히 맞추기 때문에 모든 생쥐의 우도가 1이 되고, log(1) = 0이에요.

**설명:** 비만인 생쥐에 대해 포화 모델은 "이 생쥐는 확실히 비만"이라고 예측해요 (확률 = 1). 우도 = 확률 = 1. 비만이 아닌 생쥐에 대해서는 확률 = 0이지만 우도 = 1 - 0 = 1. 모든 개별 우도가 1이고 log(1) = 0이기 때문에 전체 로그 우도 = 0 + 0 + ... = 0이에요.

</details>

**Q4.** R 출력에서 Null deviance = 80, Residual deviance = 55라면, 이탈도 차이와 자유도 차이는 어떻게 계산하나요?

<details>
<summary>정답 보기</summary>

**정답:** 이탈도 차이 = 80 - 55 = 25. 자유도 차이는 두 이탈도의 자유도 차이예요.

**설명:** 이탈도 차이 = 영 이탈도 - 잔차 이탈도 = 25. 이 값이 카이제곱 통계량이에요. 자유도는 R 출력에 함께 나오는데, 만약 Null deviance 자유도 = 299, Residual deviance 자유도 = 293이라면 차이 = 6이에요. 파라미터 6개가 추가됐다는 뜻이에요. 카이제곱 분포(자유도 6)에서 25가 얼마나 극단적인지 보면 p-값이 나와요.

</details>

### 실전 적용 문제

**문제.** 로지스틱 회귀 결과에서 다음 정보를 읽었어요.

```
Null deviance:     120.5  on 199  degrees of freedom
Residual deviance:  85.3  on 196  degrees of freedom
```

1. 이탈도 차이와 자유도 차이를 계산해보세요
2. 이것이 R² p-값 계산에서 χ²와 자유도에 해당한다는 걸 설명해보세요
3. 잔차 이탈도가 낮은 것과 높은 것 중 어느 게 더 좋은 모델인가요?

<details>
<summary>예시 답안 보기</summary>

**1. 이탈도 차이와 자유도 차이:**
```
이탈도 차이 = 120.5 - 85.3 = 35.2
자유도 차이 = 199 - 196 = 3
```

**2. χ²와 자유도로서의 의미:**
이탈도 차이 35.2가 카이제곱 통계량(χ²)이에요. 자유도 3인 카이제곱 분포에서 35.2의 위치로 p-값을 계산해요. 자유도 3은 영 모델보다 3개의 파라미터가 추가됐다는 뜻이에요(변수 3개 추가). 이 χ² = 35.2는 매우 크고 p-값은 0.001보다 훨씬 작을 거예요. 제안 모델이 영 모델보다 유의미하게 더 나은 거예요.

**3. 잔차 이탈도는 낮을수록 좋아요.**
잔차 이탈도는 제안 모델이 포화 모델(최고 모델)로부터 얼마나 이탈했는지를 나타내요. 낮을수록 제안 모델이 포화 모델에 더 가깝고, 데이터를 더 잘 설명해요. 여기서는 잔차 이탈도 85.3으로, 영 이탈도 120.5보다 낮아서 모델이 영 모델보다 확실히 더 좋아요.

> 정답이 하나가 아닐 수 있어요. 방향이 맞으면 돼요.
