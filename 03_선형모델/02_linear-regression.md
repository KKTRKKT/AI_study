## 용어 정리

| 용어 | 원문 | 한 줄 설명 |
|---|---|---|
| 선형 회귀 | Linear Regression | 직선으로 두 변수의 관계를 모델링하는 방법이에요 |
| R² (결정계수) | R-squared | 직선이 데이터 변동을 얼마나 설명하는지를 나타내요 (0~1) |
| 조정 R² | Adjusted R-squared | 변수 수를 고려해서 보정한 R²예요 |
| 분산 | Variance | 데이터가 평균으로부터 얼마나 퍼져 있는지를 나타내요 |
| F-통계량 | F-statistic | 직선이 평균보다 얼마나 더 설명력이 있는지를 비율로 나타내요 |
| p-값 | p-value | 관찰된 결과가 우연일 확률이에요. 작을수록 유의미해요 |
| 자유도 | Degrees of Freedom | 데이터에서 독립적으로 추정할 수 있는 정보의 개수예요 |

## 1. 선형 회귀의 세 가지 핵심 단계

선형 회귀는 순서가 있어요.

```
1단계: 최소제곱법으로 직선 맞추기
2단계: R² 계산하기
3단계: R²의 p-값 계산하기
```

이 세 가지가 선형 회귀의 뼈대예요. 하나씩 살펴볼게요.

---

## 2. 1단계: 최소제곱법으로 직선 맞추기

### 잠깐 복습

생쥐의 몸무게(x축)와 키(y축) 데이터가 있어요. 목표는 몸무게로 키를 예측하는 거예요.

1. 데이터에 직선을 그어요
2. 직선과 각 데이터 점의 거리(잔차)를 측정해요
3. 잔차를 제곱해서 모두 더해요(잔차 제곱합)
4. 잔차 제곱합이 최소가 되는 직선을 찾아요 → 이게 **최소제곱 직선**이에요

결과로 나온 직선:
```
예측 키 = 0.1 + 0.78 × 몸무게
```

- y절편 = 0.1 (몸무게가 0일 때 예측 키)
- 기울기 = 0.78 (몸무게가 1단위 늘면 키가 0.78 단위 늘어요)

기울기가 0이 아닌 건, 몸무게를 알면 키를 예측하는 데 **도움이 된다**는 뜻이에요.

---

## 3. 2단계: R² 계산하기

### 🤔 왜 R²가 필요한가요?

직선을 맞췄어요. 그런데 이 직선이 얼마나 **유용한지** 어떻게 알 수 있을까요?

직선이 데이터를 "얼마나 잘 설명하는가"를 0에서 1 사이 숫자로 표현하는 게 R²예요.

### R²의 직관

두 가지를 비교해요.

**평균선(기준점):**
데이터의 평균 키로 수평선을 그어요. 이 선은 몸무게 정보를 전혀 사용하지 않아요.

- 각 데이터 점과 평균선의 거리를 제곱해서 더해요
- 이걸 **SS평균(Sum of Squares around the Mean)** 이라고 해요

**최소제곱 직선:**
몸무게를 활용한 직선이에요.

- 각 데이터 점과 직선의 잔차를 제곱해서 더해요
- 이걸 **SS적합(Sum of Squares around the Fit)** 이라고 해요

최소제곱 직선이 평균선보다 잔차가 작으면, 몸무게 정보가 키 예측에 도움이 된 거예요.

### R² 공식

```
R² = (SS평균 − SS적합) / SS평균
   = (분산의 감소량) / (전체 분산)
```

**이 수식이 말하는 것:**
"직선을 사용했더니 분산이 얼마나 줄었나?" 를 비율로 나타낸 거예요.

### R² 값 해석

| R² 값 | 의미 |
|---|---|
| **0** | 직선이 아무 도움이 안 돼요. 몸무게는 키와 무관해요 |
| **0.6** | 직선이 분산의 60%를 설명해요. 몸무게가 키의 60%를 설명해요 |
| **1.0** | 직선이 분산 100%를 설명해요. 몸무게로 키를 완벽하게 예측할 수 있어요 |

### 실제 예시 3가지

**예시 1 — R² = 0.6 (일반적인 경우):**
```
SS평균 = 11.1,  SS적합 = 4.4
R² = (11.1 - 4.4) / 11.1 = 6.7 / 11.1 ≈ 0.60 (60%)
```
몸무게가 키 변동의 60%를 설명해요.

**예시 2 — R² = 1.0 (완벽한 경우):**
```
SS평균 = 11.1,  SS적합 = 0
R² = (11.1 - 0) / 11.1 = 1.0 (100%)
```
직선이 모든 데이터 점을 완벽하게 통과해요. 잔차가 0이에요.

**예시 3 — R² = 0 (쓸모없는 경우):**
```
SS평균 = 11.1,  SS적합 = 11.1
R² = (11.1 - 11.1) / 11.1 = 0 (0%)
```
직선이 평균선보다 전혀 나을 게 없어요. 몸무게가 키를 전혀 설명하지 못해요.

### 제곱합 대신 분산으로 계산해도 같은 결과

분산 = 제곱합 / n이에요. n이 분자와 분모 모두에 있어서 약분되기 때문에, 제곱합으로 계산하든 분산으로 계산하든 같은 R²가 나와요.

---

## 4. 다변수 모델에서의 R²

### 변수를 추가하면 R²가 항상 올라가요

몸무게뿐만 아니라 꼬리 길이도 함께 쓰면 어떻게 될까요?

3D 그래프가 필요해요. x축 = 몸무게, z축 = 꼬리 길이, y축 = 키. 이제 직선 대신 **평면(Plane)** 을 맞춰요.

평면의 방정식:
```
예측 키 = y절편 + 0.7 × 몸무게 + 0.5 × 꼬리 길이
```

파라미터가 3개예요 (y절편, 몸무게 계수, 꼬리 길이 계수).

**중요한 점:** 변수를 추가하면 R²가 절대 낮아지지 않아요.

왜냐하면, 최소제곱법은 쓸모없는 변수를 자동으로 파라미터를 **0으로 만들어서** 무시하거든요. 쓸모 있으면 R²가 올라가고, 쓸모 없으면 그냥 유지돼요.

> ⚠️ **문제:** 이 특성 때문에 쓸모없는 변수를 마구 추가해도 R²가 줄지 않아요. 오히려 우연히 R²가 조금 오를 수도 있어요. "동전 앞면 확률"이나 "별자리"를 변수로 넣어도 R²가 약간 올라갈 수 있어요.

### 조정 R² (Adjusted R-squared)

변수 수를 고려해서 보정한 R²예요. 변수를 추가할수록 패널티를 부여해요.

```
조정 R² = 1 - (1 - R²) × (n - 1) / (n - p - 1)
```

여기서 n은 샘플 수, p는 변수 수예요. 변수가 많아질수록 분수 값이 커져서 조정 R²가 낮아져요.

> 💡 **팁:** 다중 회귀에서는 항상 조정 R²를 함께 보고해요. R² 단독으로는 "변수를 많이 넣어서 억지로 올린 것"인지 알 수 없어요.

## 5. 3단계: R²의 p-값 계산하기

### 🤔 R²만으로는 부족해요

R² = 0.6이 나왔어요. 그런데 이게 **우연히** 그렇게 나온 건 아닐까요?

극단적인 예시로, 데이터 점이 딱 2개뿐이라면 어떤 두 점이든 그 두 점을 정확히 지나는 직선을 그릴 수 있어요. R² = 1.0이 나오지만, 이건 의미가 없어요.

데이터 점이 많아지고 R²가 여전히 높다면, 그건 우연이 아닌 진짜 관계가 있다는 증거예요.

이 "통계적 유의성"을 측정하는 게 **p-값**이에요.

### p-값 계산의 아이디어

1. **무작위 데이터**로 같은 회귀 분석을 해요 (관계가 없는 데이터)
2. 그 무작위 데이터에서 나온 R²를 기록해요
3. 이 과정을 수백만 번 반복해서 분포를 만들어요
4. 실제 데이터의 R²가 이 분포에서 얼마나 극단적인 위치에 있는지 봐요
5. "이보다 더 극단적인 값이 나올 확률" = p-값

p-값이 작으면 우연히 이런 R²가 나올 확률이 낮다는 뜻이에요. 진짜 관계가 있을 가능성이 높아요.

### F-통계량 (F-statistic)

실제로는 무작위 시뮬레이션 대신 **F-통계량**을 사용해요.

```
F = (직선이 설명하는 분산) / (직선이 설명하지 못하는 분산)
  = (SS평균 - SS적합) / (p_fit - p_mean)
    ─────────────────────────────────────────
    SS적합 / (n - p_fit)
```

**이 수식이 말하는 것:**
- 분자: "직선이 평균보다 얼마나 더 잘 설명하나?" (추가된 설명력)
- 분모: "직선으로도 설명 못 한 나머지 오차"
- F가 크면 → 직선이 잘 설명한다 → R²가 유의미하다

### 자유도(Degrees of Freedom)

공식 안의 수식들이 분산으로 변환될 때 자유도가 등장해요.

- **p_fit:** 적합 직선의 파라미터 수예요
  - 단순 직선(절편 + 기울기) → p_fit = 2
- **p_mean:** 평균선의 파라미터 수예요
  - 수평선(평균값만) → p_mean = 1

따라서 p_fit − p_mean = **추가된 파라미터 수** 예요.

> 💡 **직관:** 파라미터가 하나 더 있으면 데이터 점이 하나 더 필요해요. 자유도는 "실제로 사용 가능한 독립적인 정보의 개수"예요. 파라미터를 추정할수록 자유도가 줄어들어요.

몸무게만 쓸 때: p_fit = 2, p_mean = 1, p_fit − p_mean = **1** (추가 파라미터 1개 = 몸무게)
몸무게 + 꼬리 길이: p_fit = 3, p_mean = 1, p_fit − p_mean = **2** (추가 파라미터 2개)

### F-분포로 p-값 계산

F-통계량이 클수록 좋아요. F-분포는 자유도에 따라 모양이 달라지는 이미 알려진 확률 분포예요.

실제로는 무작위 시뮬레이션 없이, 이 F-분포에서 우리가 계산한 F값이 얼마나 극단적인 위치에 있는지 바로 계산할 수 있어요.

**F-분포의 특성:**
- 샘플이 많을수록(n − p_fit가 클수록) 분포의 꼬리가 얇아져요
- 즉, 같은 F값이라도 샘플이 많으면 p-값이 더 작아져요
- 이는 "데이터가 많을수록 같은 결과가 더 믿음직하다"는 직관과 일치해요

## 6. 좋은 결과를 위한 두 가지 조건

> 🎯 **핵심:** 선형 회귀에서 "의미 있는 결과"는 R²와 p-값 **둘 다** 좋아야 해요.

| 조건 | 좋은 수치 | 의미 |
|---|---|---|
| R² | **크게** (0에 가까울수록 나쁨) | 직선이 데이터를 잘 설명해요 |
| p-값 | **작게** (0.05 미만이 일반적 기준) | 그 설명이 우연이 아니에요 |

R²가 크지만 p-값이 크면? → 설명력이 있어 보이지만 우연일 수 있어요.
p-값이 작지만 R²가 작으면? → 통계적으로 유의미하지만 실질적 설명력이 낮아요.

---

## ⚠️ 주의할 점 / 자주 하는 실수

- **R² = 1이라도 항상 완벽한 모델이 아니에요.** 데이터가 2개뿐이면 어떤 직선이든 R² = 1이에요.
- **R²가 크다고 인과관계가 있는 건 아니에요.** 상관관계(관련이 있어 보임)와 인과관계(원인-결과)는 다른 개념이에요.
- **변수를 많이 추가하면 R²는 항상 올라가요.** 이를 해결하기 위해 다중 회귀에서는 조정 R²를 봐야 해요.
- **p-값이 작다고 효과 크기가 큰 건 아니에요.** 샘플이 매우 많으면 아주 작은 관계도 통계적으로 유의미하게 나와요. R²와 p-값을 함께 봐야 해요.
- **선형 관계가 아닌 데이터에는 선형 회귀가 잘 맞지 않아요.** 잔차 분포를 시각화해서 선형성 가정이 맞는지 확인해야 해요.

**Q1.** R²가 0.75라는 게 어떤 의미인지 설명해보세요.

<details>
<summary>정답 보기</summary>

**정답:** 사용한 변수(들)가 y변수 전체 분산의 75%를 설명한다는 뜻이에요.

**설명:** 생쥐 키 예측 예시라면, 몸무게가 키 변동의 75%를 설명해요. 나머지 25%는 몸무게로는 설명되지 않는 다른 요인들(사료 종류, 유전자 등)에 의한 변동이에요. R² = 0.75는 꽤 설명력이 높은 편이에요.

</details>

**Q2.** 왜 변수를 추가하면 R²가 줄지 않나요? 조정 R²는 이 문제를 어떻게 해결하나요?

<details>
<summary>정답 보기</summary>

**정답:** 최소제곱법은 쓸모없는 변수의 계수를 자동으로 0으로 만들어서 무시해요. 그래서 쓸모없는 변수를 추가해도 R²가 나빠지지 않아요. 심지어 우연에 의해 약간 올라갈 수도 있어요.

**설명:** 조정 R²는 변수 수가 많을수록 패널티를 부여해요. 변수를 추가했을 때 실질적인 설명력 향상이 패널티보다 크면 조정 R²가 오르고, 그렇지 않으면 내려요. 이 덕분에 "진짜 유용한 변수"만 있을 때 조정 R²가 최대가 돼요.

</details>

---

**Q3.** F-통계량의 분자와 분모가 각각 무엇을 나타내는지 설명해보세요.

<details>
<summary>정답 보기</summary>

**정답:**
- **분자:** 직선이 평균보다 추가로 설명한 분산이에요. "회귀선 덕분에 줄어든 오차"예요.
- **분모:** 회귀선으로도 설명하지 못한 나머지 분산이에요. "아직도 남은 오차"예요.

**설명:** F가 크면 분자(회귀선이 설명한 것)가 분모(설명 못 한 것)보다 크다는 뜻이에요. 직선이 의미 있게 데이터를 설명하고 있다는 증거예요. 두 변수가 무관하다면 분자와 분모가 비슷해서 F ≈ 1이 되고 p-값이 커요.

</details>

---

**Q4.** 데이터 점이 단 2개뿐일 때 R² = 1이 나오는 이유는 무엇이고, 이게 왜 문제인가요?

<details>
<summary>정답 보기</summary>

**정답:** 두 점은 항상 하나의 직선으로 완벽하게 연결되기 때문에 잔차가 0이에요. R² = 1이 나오지만 이건 아무 의미 없는 결과예요.

**설명:** R² = 1이 나오려면 직선이 모든 데이터 점을 완벽히 지나야 해요. 두 점은 기하학적으로 항상 하나의 직선으로 연결할 수 있어요. 그래서 데이터 점이 2개만 있으면 어떤 무작위 데이터라도 R² = 1이에요. 이게 p-값이 필요한 이유예요. p-값은 "이 R²가 우연히 나올 수 있는가"를 검정해줘요.

</details>

---

**Q5.** p-값이 0.001이고 R²가 0.02인 경우, 이 회귀 분석 결과를 어떻게 해석해야 할까요?

<details>
<summary>정답 보기</summary>

**정답:** 통계적으로는 유의미하지만(p < 0.05), 실질적인 설명력은 매우 낮아요(R² = 2%).

**설명:** p-값이 0.001이라는 건 이 관계가 우연이 아닐 가능성이 매우 높다는 뜻이에요. 하지만 R² = 0.02는 사용한 변수가 y값 전체 변동의 2%만 설명한다는 뜻이에요. 이런 상황은 샘플 수가 매우 많을 때 자주 발생해요. 샘플이 많으면 작은 관계도 통계적으로 유의미하게 검출돼요. "통계적으로 유의미"와 "실질적으로 의미 있음"은 다른 개념이에요.

</details>

### 실전 적용 문제

**문제.** 어떤 연구자가 "공부 시간"으로 "시험 점수"를 예측하는 선형 회귀를 했어요.

- SS평균 = 500
- SS적합 = 200
- 샘플 수 n = 30
- 파라미터 수 p_fit = 2 (y절편 + 공부시간 계수)

1. R²를 계산해보세요
2. 이 결과를 해석해보세요
3. 만약 연구자가 "수면 시간"도 변수로 추가해서 p_fit = 3이 되었고 SS적합 = 195로 줄었다면, 이 변수를 추가할 가치가 있나요? (조정 R²의 개념으로 설명해봐요)

**힌트:**
- R² = (SS평균 - SS적합) / SS평균
- 조정 R²는 변수 추가 시 패널티가 있어요
- "진짜 설명력 향상"과 "변수 추가 비용"을 비교해봐요

<details>
<summary>예시 답안 보기</summary>

**1. R² 계산:**
```
R² = (500 - 200) / 500 = 300 / 500 = 0.60 (60%)
```

**2. 해석:**
공부 시간이 시험 점수 변동의 60%를 설명해요. 꽤 높은 설명력이에요.

**3. "수면 시간" 변수 추가 여부:**

변수 추가 전: R² = 0.60 (SS적합 = 200, p_fit = 2)
변수 추가 후: R² = (500 - 195)/500 = 305/500 = 0.61 (SS적합 = 195, p_fit = 3)

R²가 0.60 → 0.61로 고작 1% 올랐어요. 파라미터는 하나 더 늘었고요. 조정 R² 계산을 하면 이 1% 향상이 패널티를 상쇄할 만큼 충분하지 않을 가능성이 높아요.

수면 시간 데이터를 수집하는 시간과 비용에 비해 설명력 향상이 미미해요. 이 경우에는 변수를 추가하지 않는 것이 나을 수 있어요. 단, p-값을 함께 확인해서 수면 시간이 통계적으로 유의미한지도 검토해야 해요.

> 정답이 하나가 아닐 수 있어요. 방향이 맞으면 돼요.

</details>

## 🎯 핵심 요약

| 개념 | 한 줄 설명 |
|---|---|
| 📐 선형 회귀 3단계 | ① 최소제곱 직선 맞추기 → ② R² 계산 → ③ p-값 계산이에요 |
| 📊 R² (결정계수) | 직선이 데이터 분산을 얼마나 설명하는지예요. 0~1 사이 값이에요 |
| 🔢 R² 공식 | (SS평균 - SS적합) / SS평균이에요 |
| ⚠️ 조정 R² | 변수 수가 많을수록 패널티를 줘서 보정한 R²예요 |
| 📈 F-통계량 | 설명한 분산 / 설명 못 한 분산의 비율이에요. 클수록 좋아요 |
| 🎲 p-값 | F-분포에서 구한 통계적 유의성이에요. 작을수록 관계가 우연이 아니에요 |
| ✅ 좋은 결과 | R²는 크고 p-값은 작아야 해요. 둘 다 좋아야 의미 있는 결과예요 |
