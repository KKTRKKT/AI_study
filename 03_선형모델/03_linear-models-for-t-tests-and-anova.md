## 1. 다중 회귀는 복잡하지 않아요

### 🤔 단순 회귀와 뭐가 다른가요?

많은 사람들이 단순 회귀와 다중 회귀를 크게 다른 것처럼 얘기해요. 하지만 실제로는 거의 같아요. 핵심 아이디어가 동일해요.

### 한 줄 정의

> **다중 회귀(Multiple Regression)란:** 변수가 여러 개인 선형 회귀예요. 직선 대신 평면(또는 그 이상)을 데이터에 맞추는 거예요.

### 일상 비유

> 💡 **비유:** 집값을 예측하고 싶어요. 면적만 보는 건 단순 회귀예요. 면적 + 방 개수 + 역까지 거리를 함께 보는 건 다중 회귀예요. 고려하는 요소가 많아졌을 뿐, 예측하는 방식의 원리는 같아요.

---

## 2. 직선에서 평면으로

### 단순 회귀: 직선 (2D)

변수 1개 (몸무게)로 변수 1개 (키)를 예측:
```
키 = y절편 + a × 몸무게
```

그래프: x축 = 몸무게, y축 = 키 → **2D 직선**

### 다중 회귀: 평면 (3D)

변수 2개 (몸무게 + 꼬리 길이)로 변수 1개 (몸통 길이)를 예측:
```
몸통 길이 = y절편 + 0.7 × 몸무게 + 0.5 × 꼬리 길이
```

그래프: x축 = 몸무게, z축 = 꼬리 길이, y축 = 몸통 길이 → **3D 평면**

> 💡 **보충 설명:** "평면에 데이터를 맞춘다"는 게 어렵게 들릴 수 있지만, 원리는 직선과 완전히 같아요. 각 데이터 점에서 평면까지의 잔차를 제곱해서 더한 값(잔차 제곱합)을 최소화하는 평면을 찾아요.

### 변수가 더 많아지면?

- 변수 3개 → 4D 공간의 초평면
- 변수 10개 → 11D 공간의 초평면

4D, 11D는 시각화할 수 없어요. 하지만 수학적 원리는 동일해요. 우리가 이해하기 어려운 건 시각화가 안 되는 것뿐이지, 계산 원리는 똑같아요.

---

## 3. R² 계산: 단순 회귀와 완전히 동일해요

### 계산 방식이 같아요

변수가 몇 개든 R² 계산 방식은 완전히 같아요.

```
R² = (SS평균 − SS적합) / SS평균
```

- **SS평균:** 평균값으로만 예측했을 때의 잔차 제곱합
- **SS적합:** 다중 회귀 모델로 예측했을 때의 잔차 제곱합

다중 회귀라도 예측 대상(몸통 길이)의 평균과 잔차만 계산하면 돼요. 변수가 몇 개인지는 관계없어요.

---

## 4. p-값 계산: 파라미터 수만 바뀌어요

### 단순 회귀 vs 다중 회귀 파라미터 수

F-통계량 계산에서 파라미터 수(p)가 달라져요.

**단순 회귀 (몸무게 하나):**
```
모델: 몸통 길이 = y절편 + a × 몸무게
p_fit = 2 (y절편, 몸무게 계수)
```

**다중 회귀 (몸무게 + 꼬리 길이):**
```
모델: 몸통 길이 = y절편 + a × 몸무게 + b × 꼬리 길이
p_fit = 3 (y절편, 몸무게 계수, 꼬리 길이 계수)
```

변수를 하나 추가할 때마다 p_fit가 1씩 늘어요.

---

## 5. 다중 회귀가 빛나는 순간 — 두 모델 비교하기

### 🤔 진짜 질문: "꼬리 길이를 추가하는 게 의미 있나?"

단순 회귀와 다중 회귀 모두 평균값(기준선)과 비교하는 것도 의미 있어요. 하지만 더 흥미로운 질문이 있어요.

"꼬리 길이를 모델에 추가하는 게 진짜 도움이 되나요? 아니면 그냥 노이즈인가요?"

이 질문에 답하려면 단순 회귀와 다중 회귀를 **서로 비교**하면 돼요.

### 단순 회귀 vs 다중 회귀 비교 방법

F-통계량을 수정해요. 평균선 대신 **단순 회귀 결과**를 기준으로 쓰면 돼요.

```
F = (SS_단순회귀 − SS_다중회귀) / (p_다중 − p_단순)
    ────────────────────────────────────────────────
                SS_다중회귀 / (n − p_다중)
```

| 항목 | 단순 회귀 (기준) | 다중 회귀 (비교 대상) |
|---|---|---|
| SS | SS_단순회귀 | SS_다중회귀 |
| 파라미터 수 | p_단순 = 2 | p_다중 = 3 |

### 해석

- F가 크고 p-값이 작으면 → "꼬리 길이가 유의미하게 도움이 돼요" → 변수 추가 가치 있음
- F가 작고 p-값이 크면 → "꼬리 길이는 추가해도 별 도움이 안 돼요" → 변수 추가 불필요

> 💡 **핵심:** 이 비교를 통해 "새로 추가한 변수가 실제로 설명력을 유의미하게 높이는가"를 통계적으로 검증할 수 있어요. 이것이 다중 회귀의 강점이에요.

---

## 6. 더 많은 변수를 추가하는 경우

### 변수를 계속 추가하면

먹이 양, 운동 시간 등을 계속 추가하면 p_fit가 계속 늘어요.

```
모델 4개 변수: y = b₀ + b₁×몸무게 + b₂×꼬리길이 + b₃×먹이양 + b₄×운동시간
p_fit = 5
```

이때 각 변수가 유의미한지 검정하려면:
1. 전체 모델 F-통계량: 이 4변수 모델이 평균보다 좋은가?
2. 개별 변수 추가 검정: 각 변수를 추가했을 때 유의미한가?

---

## ⚠️ 주의할 점 / 자주 하는 실수

- **변수를 많이 넣으면 R²는 무조건 올라가요.** 조정 R²와 p-값을 함께 확인해야 해요.
- **다중공선성(Multicollinearity)을 조심해요.** 변수 간에 강한 상관이 있으면 계수 추정이 불안정해요. 예: 몸무게와 BMI를 동시에 넣으면 문제가 생길 수 있어요.
- **변수 수가 샘플 수보다 많으면 안 돼요.** 변수가 너무 많으면 과적합이 발생해요.
- **인과관계와 혼동하지 마세요.** 다중 회귀가 유의미해도 그게 인과관계를 증명하지는 않아요.

### 개념 확인 퀴즈

**Q1.** 단순 회귀와 다중 회귀의 핵심적인 차이는 무엇인가요?

<details>
<summary>정답 보기</summary>

**정답:** 예측에 사용하는 변수의 수예요. 단순 회귀는 변수 1개로 직선을 맞추고, 다중 회귀는 변수 여러 개로 평면(또는 초평면)을 맞춰요.

**설명:** 계산 원리와 해석 방법은 동일해요. R² = (SS평균 - SS적합) / SS평균, F-통계량으로 p-값 계산 등 모든 것이 같아요. 달라지는 건 파라미터 수(p_fit)뿐이에요.

</details>

---

**Q2.** 다중 회귀에서 F-통계량의 분모가 `SS적합 / (n − p_fit)`인 이유는 무엇인가요?

<details>
<summary>정답 보기</summary>

**정답:** 파라미터가 많아질수록 자유도가 줄어들어서, 잔차 분산을 올바르게 추정하기 위해 파라미터 수를 빼줘요.

**설명:** 파라미터를 하나 추정할 때마다 데이터 포인트 "하나"가 소비되는 개념이에요. n개의 데이터가 있고 p_fit개의 파라미터를 추정하면, 자유롭게 움직일 수 있는 데이터는 n − p_fit개만 남아요. 이 자유도로 나눠야 편향 없는 분산 추정이 가능해요.

</details>

---

**Q3.** "꼬리 길이를 다중 회귀에 추가했더니 R²가 0.60에서 0.65로 올랐다"는 것만으로 꼬리 길이가 유의미한 변수라고 말할 수 있나요?

<details>
<summary>정답 보기</summary>

**정답:** 아니요. R²가 올랐더라도 반드시 p-값(통계적 유의성)을 확인해야 해요.

**설명:** R²가 올라도 샘플이 적거나 다른 이유로 그 상승이 우연일 수 있어요. 단순 회귀와 다중 회귀를 비교하는 F-통계량과 p-값으로 "꼬리 길이 추가가 통계적으로 유의미한가"를 검정해야 해요. p-값이 크면 꼬리 길이 추가는 우연에 의한 개선일 수 있어요.

</details>

---

**Q4.** 다중 회귀 모델에 "별자리"를 변수로 추가하면 R²가 어떻게 될까요? 왜 그런가요?

<details>
<summary>정답 보기</summary>

**정답:** R²가 같거나 아주 약간 올라요. 줄어들지는 않아요.

**설명:** 최소제곱법은 "별자리"가 쓸모없으면 해당 파라미터를 자동으로 0에 가깝게 설정해서 모델에서 사실상 제거해요. SS적합이 줄어들거나 유지되기 때문에 R²는 절대 내려가지 않아요. 하지만 우연히 어떤 별자리 카테고리가 데이터의 노이즈에 살짝 맞아떨어지면 R²가 아주 조금 오를 수 있어요. 이것이 조정 R²가 필요한 이유예요.

</details>

### 실전 적용 문제

**문제.** 부동산 데이터로 집값을 예측하는 모델을 만들고 있어요.

- 모델 A (단순 회귀): 집 면적만 사용. SS적합 = 800, p_fit = 2
- 모델 B (다중 회귀): 집 면적 + 역까지 거리 사용. SS적합 = 650, p_fit = 3
- SS평균 = 1000, n = 50

1. 모델 A와 B의 R²를 각각 계산해보세요
2. "역까지 거리"를 추가한 게 의미 있는지 판단하기 위한 F-통계량을 계산해보세요
3. F값이 크면 어떤 결론을 내릴 수 있나요?

**힌트:**
- R² = (SS평균 - SS적합) / SS평균
- F = [(SS_A - SS_B) / (p_B - p_A)] / [SS_B / (n - p_B)]

<details>
<summary>예시 답안 보기</summary>

**1. R² 계산:**
- 모델 A: R² = (1000 - 800) / 1000 = 200/1000 = **0.20 (20%)**
- 모델 B: R² = (1000 - 650) / 1000 = 350/1000 = **0.35 (35%)**

**2. F-통계량 계산:**
```
분자 = (SS_A - SS_B) / (p_B - p_A)
     = (800 - 650) / (3 - 2)
     = 150 / 1
     = 150

분모 = SS_B / (n - p_B)
     = 650 / (50 - 3)
     = 650 / 47
     ≈ 13.83

F = 150 / 13.83 ≈ 10.85
```

**3. 결론:**
F ≈ 10.85는 꽤 큰 값이에요. F-분포에서 이 값에 해당하는 p-값을 구하면 0.05보다 훨씬 작을 거예요(일반적으로 F > 4 정도면 p < 0.05로 봐요). 따라서 "역까지 거리" 변수를 추가하는 것이 통계적으로 유의미해요. 집 면적만 쓸 때보다 역까지 거리도 함께 고려하면 집값 예측이 유의미하게 개선돼요.

> 정답이 하나가 아닐 수 있어요. 방향이 맞으면 돼요.

</details>
