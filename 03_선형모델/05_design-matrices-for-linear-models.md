## 용어 정리

| 용어 | 원문 | 한 줄 설명 |
|---|---|---|
| 디자인 행렬 | Design Matrix | 각 데이터 포인트에 어떤 파라미터를 어떻게 적용할지 담은 표예요 |
| 절편 | Intercept | 직선이 y축을 만나는 점이에요. 디자인 행렬에서 1로 이루어진 첫 번째 열로 표현돼요 |

| 배치 효과 | Batch Effect | 서로 다른 실험실/시기에 수집된 데이터가 실제 차이 외에 갖는 체계적인 차이예요 |
| 공변량 | Covariate | 주 관심 변수 외에 모델에 포함시키는 추가 변수예요. 교란 효과를 보정할 때 써요 |

## 1. 두 가지 디자인 행렬 — 무엇이 다른가요?

### 이전 문서에서 쓴 디자인 행렬 (우리 버전)

이전 문서에서는 각 그룹에 독립적인 열을 하나씩 배정했어요.

```
이전 방식 (t검정):
┌─────────┬─────────┐
│ 대조군 열 │ 변이군 열 │
├─────────┼─────────┤
│    1    │    0    │  ← 대조군 데이터
│    1    │    0    │
│    0    │    1    │  ← 변이군 데이터
│    0    │    1    │
└─────────┴─────────┘

방정식: y = μ_대조군 + μ_변이군
```

- 열 1: 대조군 평균을 켜고 끔
- 열 2: 변이군 평균을 켜고 끔

### 표준 디자인 행렬 (현업에서 사용하는 방식)

현업에서는 조금 다른 방식을 써요.

```
표준 방식 (t검정):
┌─────────────┬────────────────┐
│ 절편 열 (모두 1) │ 변이군 오프셋 열 │
├─────────────┼────────────────┤
│      1      │       0        │  ← 대조군 데이터
│      1      │       0        │
│      1      │       1        │  ← 변이군 데이터
│      1      │       1        │
└─────────────┴────────────────┘

방정식: y = μ_대조군 + Δ(변이군 - 대조군)
```

- 열 1: 항상 1 → 대조군 평균(y절편 역할)을 항상 켜요
- 열 2: 변이군이면 1, 대조군이면 0 → "변이군이 대조군보다 얼마나 다른가"(오프셋)를 켜고 끔

### 두 방식은 수학적으로 동일해요

- **방정식 구조:** 파라미터 수가 둘 다 2개예요 (p_fit = 2)
- **잔차:** 두 방식이 동일해요
- **SS적합:** 완전히 같아요
- **F값과 p-값:** 완전히 같아요

그럼 왜 표준 방식을 쓸까요? 아마도 회귀(Regression)와의 일관성 때문이에요. 회귀의 디자인 행렬도 첫 번째 열이 1로 가득 차 있거든요.

## 2. 절편 열이 있는 표준 디자인 행렬의 의미

### 표준 방식의 해석

표준 방식에서 두 번째 열은 **"오프셋(Offset)"** 을 나타내요.

```
변이군 평균 = 대조군 평균 + Δ
Δ = 변이군 평균 - 대조군 평균
```

예시:
- 대조군 평균 = 2.2
- 변이군 평균 = 3.6
- 오프셋 Δ = 3.6 - 2.2 = **1.4**

방정식: y = 2.2 + 1.4 × (변이군이면 1, 대조군이면 0)

이 방식은 "변이군이 대조군과 얼마나 차이가 나는가"를 직접적으로 나타내줘요. 이 차이의 통계적 유의성을 검정하는 게 t검정의 핵심이에요.

## 3. 선형 회귀의 디자인 행렬

### 구조

선형 회귀에서도 같은 디자인 행렬 구조를 써요.

```
선형 회귀 디자인 행렬:
┌─────────────┬──────────────────┐
│ 절편 열 (모두 1) │ x값 열 (연속형 숫자) │
├─────────────┼──────────────────┤
│      1      │       x₁         │
│      1      │       x₂         │
│      1      │       x₃         │
│      1      │       x₄         │
└─────────────┴──────────────────┘
```

- **열 1:** 모두 1 → y절편(β₀)을 위한 열
- **열 2:** 각 데이터 포인트의 x값 → 기울기(β₁)를 스케일링

방정식:
```
y = β₀ × 1 + β₁ × x
  = β₀ + β₁ × x
```

### 직접 계산해보기

y절편 β₀ = 0.01, 기울기 β₁ = 0.8이라면:

첫 번째 데이터 포인트(x₁ = 0.5):
```
예측값 = 0.01 × 1 + 0.8 × 0.5 = 0.01 + 0.40 = 0.41
```

두 번째 데이터 포인트(x₂ = 1.2):
```
예측값 = 0.01 × 1 + 0.8 × 1.2 = 0.01 + 0.96 = 0.97
```

모든 데이터 포인트에 대해 이렇게 예측값을 계산하면 직선 위의 점들이 나와요.

> 💡 **보충 설명:** 디자인 행렬과 파라미터 벡터의 곱이 예측값을 만들어요. 이걸 행렬 곱셈(Matrix Multiplication)이라고 해요. 컴퓨터는 이 행렬 연산을 아주 효율적으로 처리해요.

## 4. t검정 + 회귀 동시에 하기

### 🤔 때로는 두 가지를 동시에 해야 해요

일반 대조군 생쥐와 특정 유전자가 변이된 생쥐가 있어요. 두 그룹 모두 몸무게-키 관계를 따르지만, 변이 생쥐는 같은 몸무게에서도 키가 더 커요.

이 경우 두 가지를 동시에 알고 싶어요:
1. 몸무게와 키의 관계 (회귀)
2. 변이 생쥐가 대조군보다 유의미하게 큰가 (t검정)

단순 회귀만 하면 두 그룹의 차이를 무시해요.
단순 t검정만 하면 몸무게의 영향을 무시해요.
둘을 동시에 해야 올바른 분석이에요.

### 결합 디자인 행렬

```
결합 디자인 행렬:
┌─────────────┬─────────────────┬──────────────┐
│ 절편 열 (모두 1) │ 변이군 오프셋 열 │ 몸무게(x값) 열 │
├─────────────┼─────────────────┼──────────────┤
│      1      │        0        │     x₁       │  ← 대조군 데이터
│      1      │        0        │     x₂       │
│      1      │        1        │     x₃       │  ← 변이군 데이터
│      1      │        1        │     x₄       │
└─────────────┴─────────────────┴──────────────┘
```

방정식:
```
y = β₀(y절편) + β₁(변이군 오프셋) × (변이군이면 1, 아니면 0) + β₂(기울기) × 몸무게
```

이 모델은:
- 대조군 직선: y = β₀ + β₂ × 몸무게
- 변이군 직선: y = (β₀ + β₁) + β₂ × 몸무게

두 직선은 **같은 기울기**를 가지지만 y절편이 달라요. β₁이 변이 생쥐의 "추가 크기"를 나타내요.

### p_fit = 3 (파라미터 3개)

이 모델에는 파라미터가 3개예요:
- β₀: 대조군 y절편
- β₁: 변이군 오프셋
- β₂: 기울기 (두 그룹 공통)

### 다양한 비교 가능

이 결합 모델을 단순 모델과 비교해서 각 요소의 중요성을 검정할 수 있어요.

| 비교 | 의미 |
|---|---|
| 결합 모델 vs 전체 평균 | 몸무게 + 그룹 유형이 전혀 없는 것보다 좋은가? |
| 결합 모델 vs 단순 회귀 | 그룹 유형을 추가하면 유의미하게 좋아지나? |
| 결합 모델 vs 단순 t검정 | 몸무게를 추가하면 유의미하게 좋아지나? |

각 비교에서 F 값이 크고 p-값이 작으면 해당 요소가 의미 있다는 증거예요.

> 💡 **강의 결과:** 결합 모델과 전체 평균을 비교하면 F = 21.88, p = 0.003이에요. 그룹 유형과 몸무게를 함께 고려하는 게 통계적으로 유의미하게 더 좋아요.

## 5. 배치 효과(Batch Effect) 보정하기

### 🤔 배치 효과가 뭔가요?

실험실 A가 먼저 데이터를 수집하고, 실험실 B가 나중에 같은 실험을 반복했어요. 그런데 실험실 B의 측정값이 전체적으로 실험실 A보다 작아요. 이건 생물학적 차이가 아니라 **측정 환경의 차이** 때문이에요.

이런 체계적 차이를 **배치 효과(Batch Effect)** 라고 해요.

두 실험실의 데이터를 합쳐서 분석하려면 이 배치 효과를 보정해야 해요.

### 배치 효과 보정 디자인 행렬

```
보정 디자인 행렬:
┌─────────────┬─────────────────┬─────────────────────────┐
│ 절편 열 (모두 1) │ 실험실B 오프셋 열 │ 변이 효과(대조군 vs 변이군) 열 │
├─────────────┼─────────────────┼─────────────────────────┤
│      1      │        0        │           0             │  ← A실험실 대조군
│      1      │        0        │           1             │  ← A실험실 변이군
│      1      │        1        │           0             │  ← B실험실 대조군
│      1      │        1        │           1             │  ← B실험실 변이군
└─────────────┴─────────────────┴─────────────────────────┘
```

방정식:
```
y = β₀(A실험실 대조군 기준값) + β₁(B실험실 오프셋) + β₂(변이군 효과)
```

이 모델에서:
- β₁: B실험실과 A실험실의 체계적 차이 (배치 효과)를 흡수해요
- β₂: 배치 효과를 보정한 후의 순수한 변이 효과예요

### 검정 방법

이 모델과 배치 효과만 있는 단순 모델을 비교해요.

```
단순 모델: y = β₀ + β₁(B실험실 오프셋)
결합 모델: y = β₀ + β₁(B실험실 오프셋) + β₂(변이군 효과)
```

F 값이 크고 p-값이 작으면 → 배치 효과를 보정하고도 대조군과 변이군의 차이가 유의미해요.

> 💡 **왜 중요한가요?** 배치 효과를 무시하고 분석하면 실험실 차이와 생물학적 차이가 뒤섞여서 잘못된 결론이 나올 수 있어요. 디자인 행렬에 배치 변수를 포함시키면 이 혼란을 명확하게 분리할 수 있어요.

## 6. 디자인 행렬 정리

### 핵심 원칙

디자인 행렬의 각 열은 하나의 **파라미터**에 대응해요.

| 열의 내용 | 파라미터의 역할 |
|---|---|
| 모두 1 | y절편 (기준값) |
| 0/1 (그룹 구분) | 특정 그룹의 오프셋 |
| 연속형 숫자 (x값) | 기울기 |
| 0/1과 숫자 혼합 | 회귀 + t검정 동시 처리 |

> 💡 **핵심:** 디자인 행렬에 어떤 숫자든 넣을 수 있어요. 0과 1만 넣으면 범주형 데이터(t검정, ANOVA), 연속형 숫자를 넣으면 회귀, 둘을 혼합하면 두 분석을 동시에 할 수 있어요.

### 일반 규칙

현업에서 쓰는 표준 디자인 행렬은 항상 **첫 번째 열이 모두 1** 이에요.
이후 열들은 분석 목적에 따라 달라져요.

---

## ⚠️ 주의할 점 / 자주 하는 실수

- **절편 열을 빼면 안 돼요.** 표준 디자인 행렬의 첫 번째 열(모두 1)을 제거하면 모델이 원점(0, 0)을 통과하도록 강제돼요. 대부분의 경우 이건 맞지 않아요.
- **범주형 변수를 그냥 숫자(1, 2, 3...)로 인코딩하지 마세요.** "대조군=1, 변이군=2"처럼 쓰면 그룹 간 거리(1과 2 사이)가 의미 있는 것처럼 해석될 수 있어요. 올바른 방법은 0/1로 구성된 열을 쓰는 거예요.
- **기준 그룹을 잘 설정해야 해요.** 표준 방식에서 첫 번째 열의 파라미터가 기준 그룹을 나타내요. 어떤 그룹을 기준으로 쓰느냐에 따라 오프셋 파라미터의 해석이 달라져요.
- **배치 효과를 무시하지 마세요.** 서로 다른 환경에서 수집된 데이터를 합칠 때는 배치 효과를 반드시 고려해야 해요.

## 🧪 연습해보기

### 개념 확인 퀴즈

**Q1.** 표준 디자인 행렬에서 첫 번째 열이 항상 1인 이유는 무엇인가요?

<details>
<summary>정답 보기</summary>

**정답:** 첫 번째 열의 1들은 y절편(기준값, 절편) 파라미터에 대응해요. 모든 데이터 포인트에서 y절편이 적용되기 때문에 항상 1이에요.

**설명:** y = β₀ + β₁x에서 β₀는 모든 x에 대해 항상 더해지는 상수예요. 이걸 디자인 행렬로 표현하면 "β₀를 곱하는 값 = 1"이 모든 행에 있어야 해요. 첫 번째 열의 1들이 바로 "β₀를 항상 켜두는" 역할이에요.

</details>

---

**Q2.** 두 가지 t검정 디자인 행렬(우리 버전 vs 표준 버전)이 수학적으로 동일한 결과를 내는 이유는 무엇인가요?

<details>
<summary>정답 보기</summary>

**정답:** 두 방식 모두 같은 수의 파라미터(p_fit = 2)를 추정하고, 잔차가 완전히 동일하기 때문이에요.

**설명:** 우리 버전은 "대조군 평균, 변이군 평균" 두 파라미터를 직접 추정해요. 표준 버전은 "대조군 평균, 두 그룹 간 차이" 두 파라미터를 추정해요. 두 파라미터 세트는 서로 변환이 가능해요. SS적합, p_fit가 동일하기 때문에 F값과 p-값도 같아요.

</details>

**Q3.** "회귀 + t검정 동시" 모델에서 변이군 오프셋 파라미터(β₁)가 의미하는 것은 무엇인가요?

<details>
<summary>정답 보기</summary>

**정답:** 같은 몸무게에서 변이 생쥐가 대조군 생쥐보다 얼마나 더 큰지를 나타내요.

**설명:** 결합 모델에서 두 그룹은 같은 기울기를 가지지만 y절편이 달라요. β₁은 변이군 직선이 대조군 직선보다 y축에서 얼마나 위에 있는지예요. "몸무게를 통제했을 때 변이 생쥐의 추가적인 크기"를 나타내요. 이게 β₁에 대한 p-값이 작으면 변이 효과가 통계적으로 유의미하다는 뜻이에요.

</details>

**Q4.** 배치 효과를 무시하고 두 실험실 데이터를 합쳐서 분석하면 어떤 문제가 생기나요?

<details>
<summary>정답 보기</summary>

**정답:** 실험실 간 체계적 차이(배치 효과)가 생물학적 차이(변이 효과)와 뒤섞여서 잘못된 결론이 날 수 있어요.

**설명:** 예를 들어, 실험실 B의 측정값이 전체적으로 낮다면, B실험실의 변이군 데이터가 A실험실의 대조군과 비슷하게 보일 수 있어요. 이러면 "변이 효과 없음"이라는 잘못된 결론이 나올 수 있어요. 디자인 행렬에 배치 변수를 포함하면 이 실험실 차이를 모델이 흡수하고, 나머지 파라미터가 순수한 생물학적 효과를 반영해요.

</details>

**Q5.** 디자인 행렬에서 연속형 숫자와 0/1을 함께 쓰는 경우의 예시를 들고, 이게 어떤 분석을 가능하게 하는지 설명해보세요.

<details>
<summary>정답 보기</summary>

**정답:** 회귀와 t검정을 동시에 수행할 수 있어요.

**설명:** 예: 몸무게(연속형) + 변이/대조군(0/1) 두 열을 가진 디자인 행렬을 쓰면, 몸무게와 크기의 관계(회귀)와 동시에 그룹 간 차이(t검정)를 하나의 분석으로 처리할 수 있어요. 이 방법은 "몸무게를 통제한 후의 그룹 차이"를 더 정확하게 측정해줘요. 단순 t검정은 몸무게 영향을 무시하기 때문에 부정확할 수 있어요.

</details>

### 실전 적용 문제

**문제.** 대학생들의 공부 시간과 수면 시간이 시험 점수에 미치는 영향을 분석하려고 해요.

추가로 두 그룹(이공계 전공 vs 인문계 전공)이 있어요.

1. 이 분석을 위한 디자인 행렬의 구조를 설계해보세요 (열 수, 각 열의 내용)
2. 각 파라미터가 무엇을 의미하는지 설명해보세요
3. "전공에 따른 차이가 유의미한가"를 검정하려면 어떤 두 모델을 비교해야 하나요?

**힌트:**
- 첫 번째 열은 항상 1이에요 (y절편)
- 범주형 변수(전공)는 0/1 열로 표현해요
- 연속형 변수(공부 시간, 수면 시간)는 실제 숫자로 표현해요

<details>
<summary>예시 답안 보기</summary>

**1. 디자인 행렬 구조 (열 4개):**

```
┌──────┬──────────────────┬──────────────────┬────────────────┐
│ 절편 │ 이공계(0) vs 인문계(1) │ 공부 시간 (연속형) │ 수면 시간 (연속형) │
├──────┼──────────────────┼──────────────────┼────────────────┤
│  1   │        0         │      공부시간₁    │   수면시간₁     │  ← 이공계
│  1   │        0         │      공부시간₂    │   수면시간₂     │
│  1   │        1         │      공부시간₃    │   수면시간₃     │  ← 인문계
│  1   │        1         │      공부시간₄    │   수면시간₄     │
└──────┴──────────────────┴──────────────────┴────────────────┘
```

p_fit = 4

**2. 각 파라미터의 의미:**
- β₀ (절편): 이공계 학생이 공부 시간 = 0, 수면 시간 = 0일 때 예상 점수 (기준값)
- β₁ (전공 오프셋): 동일한 공부·수면 시간 조건에서 인문계가 이공계보다 얼마나 더/덜 받는지
- β₂ (공부 시간 기울기): 공부 시간이 1시간 늘 때 점수 변화
- β₃ (수면 시간 기울기): 수면 시간이 1시간 늘 때 점수 변화

**3. 전공 효과 검정:**

- **결합 모델 (p_fit = 4):** 절편 + 전공 + 공부시간 + 수면시간
- **단순 모델 (p_fit = 3):** 절편 + 공부시간 + 수면시간 (전공 열 제거)

이 두 모델을 F-통계량으로 비교해요. p-값이 작으면 "공부·수면 시간을 통제해도 전공에 따른 차이가 유의미하다"는 결론이에요.

> 정답이 하나가 아닐 수 있어요. 방향이 맞으면 돼요.

</details>
