# 📊 회귀분석 핵심 통계 개념 정리

> Coefficient · Std. Error · t-statistic · p-value

---

## 🔗 네 개념의 관계 한눈에 보기

```
Coefficient  ÷  Std. Error  =  t-statistic  →  p-value  →  최종 판단
   효과 크기      불확실성        신호/잡음 비율     우연일 확률
```

---

## 1. Coefficient (계수)

### 정의
> **입력 변수가 결과에 미치는 영향의 크기와 방향을 나타내는 숫자**

### 세 가지 역할

| 역할 | 의미 | 예시 |
|------|------|------|
| **크기** | 영향력이 얼마나 강한가 | 0.046 vs 0.189 → 라디오가 더 강함 |
| **방향** | 결과를 올리는가(+), 내리는가(-) | +0.046 → 증가, -0.001 → 감소 |
| **순수 효과** | 다른 변수 고정 후 이 변수만의 효과 | 다중회귀에서 각 변수의 독립적 영향 |

### 실제 예시

```
판매량 = 2.939 + 0.046×TV + 0.189×라디오 - 0.001×신문

TV coefficient = +0.046
→ TV 광고비 1 증가 시 판매량 0.046 증가

신문 coefficient = -0.001
→ 신문 광고비 1 증가 시 판매량 0.001 감소
```

### ⚠️ 주의
Coefficient 숫자만으로는 **신뢰 여부를 판단할 수 없음** → Std. Error와 함께 봐야 함

---

## 2. Std. Error (표준오차)

### 정의
> **추정한 Coefficient가 얼마나 불안정한가 (불확실성의 크기)**

### 핵심 아이디어

우리가 가진 데이터는 전체 중 **일부 샘플**이기 때문에, 데이터를 새로 수집하면 Coefficient가 조금씩 달라집니다. 이 **달라지는 정도**가 Std. Error입니다.

```
Std. Error 작음 → 어떤 샘플이어도 비슷한 결과 → 신뢰 가능 ✅
Std. Error 큼   → 샘플마다 결과가 크게 달라짐 → 신뢰 어려움 ⚠️
```

### Std. Error에 영향을 주는 요소

| 요소 | 영향 |
|------|------|
| 데이터 수(n) 많을수록 | Std. Error ↓ (안정적) |
| 데이터 분산 클수록 | Std. Error ↑ (불안정) |
| 변수 간 상관관계 높을수록 | Std. Error ↑ (다중공선성) |

### 실제 예시

```
신문 coefficient = -0.001,  Std. Error = 0.0059

-0.001 ± 0.0059
→ 실제 값이 +0.0049 ~ -0.0069 사이일 수 있음
→ 양수도 가능! → 효과 방향조차 확신 불가 ❌

TV coefficient = 0.046,  Std. Error = 0.0014

0.046 ± 0.0014
→ 실제 값이 0.0446 ~ 0.0474 사이
→ 항상 양수! → 효과 있음을 확신 가능 ✅
```

## 3. t-statistic (t-통계량)

### 정의
> **Coefficient를 Std. Error로 나눈 값 — "신호 대 잡음 비율"**

```
t-statistic = Coefficient ÷ Std. Error
```

### 핵심 질문
> "이 변수의 실제 효과가 0일 수도 있지 않을까?"

t-statistic은 이 가능성을 수치로 검증합니다.

### t값 크기별 해석

| t값 (절댓값) | 의미 |
|-------------|------|
| < 1 | 신호보다 잡음이 큼 → 거의 유의미하지 않음 ❌ |
| 1 ~ 2 | 경계선, 주의 필요 ⚠️ |
| > 2 | 신호가 잡음의 2배 → 유의미할 가능성 높음 ✅ |
| > 3 | 꽤 확실한 효과 ✅✅ |
| > 10 | 매우 강력하고 확실한 효과 ✅✅✅ |

### 실제 예시

```
TV:   t = 0.046  ÷ 0.0014 = 32.8  → 신호가 잡음의 32.8배 → 확실한 효과 ✅
신문: t = -0.001 ÷ 0.0059 = -0.18 → 잡음이 신호보다 훨씬 큼 → 효과 불분명 ❌
```

### t-statistic → p-value 변환 원리

```
귀무가설(효과 = 0) 하에서 t값의 분포:

    ┌──────────────┐
  ──┘               └──
-3   -2   -1   0   1   2   3

t = 32.8 → 이 분포에서 극히 드문 값 → p-value ≈ 0
t = 0.18 → 이 분포에서 매우 흔한 값 → p-value ≈ 0.86
```

## 4. p-value

### 정의
> **"이 변수의 진짜 효과가 0인데도, 우연히 지금처럼 극단적인 결과가 나올 확률"**

### 판단 기준

| p-value | 해석 | 결론 |
|---------|------|------|
| < 0.001 | 우연일 확률 0.1% 미만 | ✅✅✅ 매우 강력한 근거 |
| < 0.01 | 우연일 확률 1% 미만 | ✅✅ 강한 근거 |
| < 0.05 | 우연일 확률 5% 미만 | ✅ 유의미 (일반적 기준) |
| < 0.10 | 우연일 확률 10% 미만 | ⚠️ 약한 근거, 참고만 |
| ≥ 0.10 | 우연일 확률 10% 이상 | ❌ 근거 없음 |

> 💡 **빠른 직관:** p-value × 100 = "우연일 확률 몇 %"
> - p = 0.03 → 3% 우연 → 믿을 만함
> - p = 0.86 → 86% 우연 → 못 믿음

### 흔한 오해 3가지

| 오해 | 올바른 해석 |
|------|------------|
| "p = 0.03이면 효과 있을 확률 97%" | ❌ p-value는 효과의 존재 확률이 아님 |
| "p가 작으면 효과가 크다" | ❌ 효과의 크기는 Coefficient를 봐야 함 |
| "p > 0.05이면 효과가 없다" | ❌ "효과 없음 증명"이 아닌 "효과 있음을 증명 못 함" |

## 5. 네 지표 통합 해석 — 광고 데이터 예시

```
판매량 = 2.939 + 0.046×TV + 0.189×라디오 - 0.001×신문
```

| 변수 | Coefficient | Std. Error | t-statistic | p-value | 결론 |
|------|------------|------------|-------------|---------|------|
| 절편 | 2.939 | 0.3119 | 9.42 | <0.0001 | ✅ |
| TV | 0.046 | 0.0014 | 32.8 | <0.0001 | ✅ |
| 라디오 | 0.189 | 0.0086 | 22.0 | <0.0001 | ✅ |
| 신문 | -0.001 | 0.0059 | -0.18 | 0.8599 | ❌ |

### 분석 결과 읽는 순서

```
① Coefficient  → "TV 광고비가 늘면 판매량이 0.046 증가"
② Std. Error   → "이 추정에는 ±0.0014 수준의 불확실성 존재"
③ t-statistic  → "효과(0.046)가 불확실성(0.0014)의 32.8배"
④ p-value      → "효과가 없는데 이런 결과가 나올 확률 0.01% 미만 → 믿을 수 있음 ✅"
```

## 6. 전체 흐름 최종 요약

```
Coefficient
  → 효과의 방향(+/-) 과 크기를 나타냄
  → 단독으로는 신뢰 여부 판단 불가

Std. Error
  → coefficient의 불확실성을 나타냄
  → 작을수록 안정적, 클수록 불안정

t-statistic = Coefficient ÷ Std. Error
  → 신호(효과) 대 잡음(불확실성) 비율
  → 클수록 효과가 우연이 아닐 가능성 높음

p-value (t-분포에서 계산)
  → 효과가 없다는 가정 하에 이런 결과가 우연일 확률
  → 0.05 미만이면 통계적으로 유의미

최종 판단:
p < 0.05 → 이 변수는 결과에 유의미한 영향을 준다 ✅
p ≥ 0.05 → 이 변수의 효과를 확신하기 어렵다 ❌
```
