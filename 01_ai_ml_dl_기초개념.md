# 🤖 AI / ML / DL 기초 개념

---

## 1. AI, ML, DL이란?

### 🧠 AI (Artificial Intelligence, 인공지능)

> **한 줄 정의:** 컴퓨터가 스스로 생각하고, 판단하고, 행동하도록 만든 시스템

AI는 주어진 **환경이나 데이터**를 바탕으로 다음과 같은 능력을 갖춘 시스템입니다:

- **인지** → 데이터를 감지하고 파악함
- **학습** → 경험을 통해 더 나아짐
- **추론** → 논리적으로 판단함
- **목표 달성** → 예측, 행동 선택, 계획을 통해 목표를 이룸

📌 **예시:** 체스를 두는 AI, 음성을 인식하는 시리(Siri), 자율주행 자동차

---

### 📊 ML (Machine Learning, 머신러닝)

> **한 줄 정의:** AI를 구현하는 방법 중 하나로, 데이터로부터 스스로 학습하는 방법론

- AI의 **하위 개념**입니다.
- 사람이 일일이 규칙을 프로그래밍하지 않아도, **데이터를 보여주면 스스로 규칙을 학습**합니다.

📌 **예시:** 수만 장의 고양이 사진을 학습해서 "이건 고양이다"라고 스스로 판단하는 모델

---

### 🧬 DL (Deep Learning, 딥러닝)

> **한 줄 정의:** 머신러닝 중에서도 **신경망(Neural Network)**을 사용하는 학습 방법

- ML의 **하위 개념**입니다.
- 인간의 뇌 구조에서 영감을 받아, 여러 층의 뉴런(신경세포)으로 이루어진 구조를 사용합니다.

📌 **예시:** ChatGPT, 이미지 인식, 음성 합성 등 현대 AI의 핵심 기술

---

### 관계 요약

```
AI (인공지능)
└── ML (머신러닝)
    └── DL (딥러닝)
```

**AI가 가장 큰 개념이고, 그 안에 ML이, 그 안에 DL이 포함됩니다.**

---

## 2. Feature와 Label

머신러닝에서 가장 중요한 두 가지 개념입니다.

---

### 📥 Feature (피처, 특성)

> **모델이 예측에 사용하는 입력 정보. 예측의 근거/단서**

- 모델에게 "이런 정보가 있어"라고 알려주는 **입력값**
- 예측 함수 `f`의 **input**으로 들어가는 변수들

📌 **예시: 집값 예측 모델**
- Feature (입력): 지역, 평수, 방 개수, 연식, 층수 ...
- 이 정보들을 보고 → 집값을 예측

---

### 🎯 Label (라벨, 목표값)

> **모델이 예측하려는 정답. 학습의 목표값**

- 모델이 맞춰야 할 **정답**
- 학습 중에는 라벨을 알고 있고, 실제 예측 때는 모르는 값

📌 **예시: 집값 예측 모델**
- Label (정답): 실제 집값 (5억, 8억, 12억 ...)

---

### Feature vs Label 비교

| 구분 | Feature (특성) | Label (라벨) |
|------|---------------|-------------|
| 역할 | 입력 (Input) | 출력/정답 (Output) |
| 시점 | 예측 시 알고 있음 | 학습 시에만 알고 있음 |
| 예시 (집값) | 지역, 평수, 연식 | 실제 집값 |
| 예시 (스팸 분류) | 이메일 내용, 발신자 | 스팸/정상 |

---

## 3. 1D Feature 기반 학습 (가장 단순한 형태)

Feature가 **딱 하나**일 때의 머신러닝을 이해하면 전체 개념이 잡힙니다.

> **핵심 목표:** Feature와 Label 사이의 관계를 잘 설명하는 함수 `f`를 찾는 것

---

### 📈 학습이란?

입력(Feature) → 출력(Label)의 관계를 찾는 과정입니다.

- 예를 들어, "공부 시간(Feature)"이 늘어날수록 "시험 점수(Label)"가 올라가는 관계를 수식으로 표현하는 것
- 하지만 이 관계를 표현하는 함수는 **무수히 많습니다** → 어떤 함수가 제일 좋을까?

---

### 🗂️ 가설 공간 (Hypothesis Space)

> **관계를 표현할 수 있는 모든 후보 함수들의 집합 (집합 F)**

- 직선일 수도 있고, 곡선일 수도 있고, 복잡한 수식일 수도 있습니다.
- 이 후보들 중에서 **가장 데이터를 잘 설명하는 함수**를 학습을 통해 고릅니다.

---

### 🔧 모델 (Model)

> **가설 공간 F에 속한 특정 하나의 함수 `f`**

- 학습이 끝나면 "이 함수가 제일 적합하다"고 선택된 결과가 바로 **모델**입니다.
- 모델이 잘 학습되면 → **새로운 입력 x에 대해서도 y를 예측**할 수 있습니다.

---

### 왜 f를 학습하는가?

잘 학습된 함수 `f`가 있다면:

1. **새로운 입력에 대한 예측 가능** → 처음 보는 데이터에도 결과를 예측할 수 있음
2. **각 Feature의 중요도 파악 가능** → "어떤 요소가 결과에 얼마나 영향을 미치는가?"
3. **구조 이해 가능** → 단순 예측을 넘어 데이터의 본질적인 관계를 이해할 수 있음

📌 **예시:** 집값 예측 모델이 잘 학습되면 → "평수보다 지역이 집값에 더 큰 영향을 미친다"는 인사이트를 얻을 수 있음

---

## 4. 정리

```
데이터 (Feature + Label)
       ↓
   모델 학습 (가설 공간에서 최적의 f를 찾음)
       ↓
  모델 (최적의 함수 f)
       ↓
새로운 Feature 입력 → Label 예측
```

> 머신러닝의 핵심은 결국 **"데이터로부터 패턴을 찾아, 새로운 데이터에도 적용할 수 있는 함수를 학습하는 것"**입니다.
